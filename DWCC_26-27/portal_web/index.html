Metodología

Cómo vamos a trabajar en DWCC

No se trata de memorizar instrucciones: se trata de entender cómo piensa una página web, cómo responde el navegador y cómo depurar cuando algo falla.

📘

Apuntes

Base conceptual, ejemplos progresivos y errores frecuentes para entender antes de copiar.

🧩

Ejercicios

Pequeños retos para practicar sintaxis, razonamiento y depuración paso a paso.

🛠️

Prácticas guiadas

Construcción dirigida de funcionalidades reales: formularios, DOM, datos y APIs.

🚀

Prácticas globales

Cierres de bloque para integrar todo lo aprendido y demostrar autonomía técnica.

Primer día, primera decisión

Abre el editor. Abre el navegador. Empieza a probar.

La web se aprende escribiendo código, rompiéndolo, leyendo errores y reconstruyendo mejor.
