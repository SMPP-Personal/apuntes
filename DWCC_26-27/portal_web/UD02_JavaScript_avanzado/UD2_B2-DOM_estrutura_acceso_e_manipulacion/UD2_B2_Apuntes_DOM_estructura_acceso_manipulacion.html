UD2_B2. DOM: estructura, acceso y manipulación

Introducción al DOM del navegador: selección, lectura, modificación, creación y eliminación de elementos HTML mediante JavaScript.

1. Qué vamos a aprender en este bloque

En este bloque empezamos a trabajar con una de las partes más importantes de JavaScript en el navegador: el DOM.

Hasta ahora hemos escrito programas que trabajan principalmente con variables, funciones, arrays, objetos y datos. A partir de este bloque, JavaScript empezará a interactuar con una página HTML real.

En este documento aprenderás a:

  • entender qué es el DOM;
  • acceder a elementos de una página con JavaScript;
  • leer y modificar texto y contenido HTML;
  • cambiar atributos y clases CSS;
  • crear elementos nuevos desde JavaScript;
  • añadir elementos a la página;
  • eliminar elementos;
  • generar contenido dinámico a partir de arrays y objetos.

Importante: en este bloque todavía no trabajaremos eventos, formularios ni asincronía. Primero necesitamos aprender a acceder y modificar la estructura de la página.

2. Qué es el DOM y el objeto document

El DOM significa Document Object Model, o modelo de objetos del documento.

Cuando el navegador carga una página HTML, no se limita a mostrar el texto del fichero. Internamente construye una estructura en forma de árbol, donde cada etiqueta HTML se convierte en un objeto que JavaScript puede consultar y modificar.

Por ejemplo, este HTML:

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Ejemplo DOM</title>
</head>
<body>
  <h1>Mi página</h1>
  <p>Primer párrafo</p>
</body>
</html>

El navegador lo interpreta como una estructura de objetos parecida a esta:

document
└── html
    ├── head
    │   ├── meta
    │   └── title
    └── body
        ├── h1
        └── p

La idea clave es esta:

El DOM permite que JavaScript acceda a una página HTML y la modifique mientras se está ejecutando en el navegador.

2.1. El objeto document

El punto de entrada principal al DOM es el objeto document.

document representa la página cargada en el navegador.

Con document podemos:

  • buscar elementos;
  • leer contenido;
  • modificar contenido;
  • crear elementos nuevos;
  • insertar elementos;
  • eliminar elementos.

Ejemplo:

console.log(document);

Si ejecutas esa instrucción en la consola del navegador, verás una representación del documento HTML cargado.

Importante

El DOM no forma parte del núcleo del lenguaje JavaScript: es una API proporcionada por el navegador. Por eso document, querySelector() o createElement() no están disponibles directamente al ejecutar un archivo con Node.js.

Los ejemplos de este bloque deben ejecutarse dentro de una página HTML abierta en el navegador. Si intentas ejecutar directamente node script.js, aparecerá un error similar a:

ReferenceError: document is not defined

3. Preparar correctamente HTML y JavaScript

Para que JavaScript pueda acceder a los elementos de la página, el HTML debe estar cargado antes de ejecutar el código.

Una forma sencilla es colocar el script al final del body:

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Prueba DOM</title>
</head>
<body>
  <h1 id="titulo">Hola DOM</h1>
  <p class="mensaje">Primer mensaje</p>

  <script src="script.js"></script>
</body>
</html>

Otra forma habitual es usar defer en el enlace al fichero JavaScript:

<script src="script.js" defer></script>

En este bloque usaremos preferentemente una de estas dos opciones.

Error frecuente: enlazar el JavaScript en el head sin defer y luego intentar acceder a elementos que todavía no se han cargado.

4. Seleccionar elementos del DOM

Para modificar una página, primero necesitamos seleccionar el elemento sobre el que queremos trabajar.

Los métodos más habituales son:

  • document.querySelector();
  • document.querySelectorAll().

4.1. querySelector(): seleccionar un único elemento

querySelector() devuelve el primer elemento que coincide con un selector CSS.

Sintaxis:

const elemento = document.querySelector("selectorCSS");

Ejemplo de HTML:

<h1 id="titulo">Bienvenido/a</h1>
<p class="mensaje">Este es el primer mensaje.</p>
<p class="mensaje">Este es el segundo mensaje.</p>

Ejemplo de JavaScript:

const titulo = document.querySelector("#titulo");
const primerMensaje = document.querySelector(".mensaje");

console.log(titulo);
console.log(primerMensaje);

Observa los selectores:

Selector Qué selecciona
"h1" El primer <h1> que encuentre
"#titulo" El elemento con id="titulo"
".mensaje" El primer elemento con class="mensaje"
"section p" El primer <p> dentro de un <section>

En una página HTML correcta, cada valor de id debe ser único.

4.2. querySelectorAll(): seleccionar varios elementos

querySelectorAll() devuelve todos los elementos que coinciden con un selector CSS.

Ejemplo de HTML:

<ul>
  <li class="tarea">Estudiar JavaScript</li>
  <li class="tarea">Practicar DOM</li>
  <li class="tarea">Repasar arrays</li>
</ul>

Ejemplo de JavaScript:

const tareas = document.querySelectorAll(".tarea");

console.log(tareas);
console.log(tareas.length);

El resultado no es un array normal, sino una NodeList. En el caso de querySelectorAll(), esa NodeList es estática: contiene los elementos que coincidían en el momento de realizar la selección. Si después se añaden nuevos elementos al DOM, la colección no se actualiza automáticamente.

Aun así, podemos recorrerla con forEach:

const tareas = document.querySelectorAll(".tarea");

tareas.forEach((tarea) => {
  console.log(tarea.textContent);
});

Recuerda: querySelector() selecciona un elemento. querySelectorAll() selecciona varios.

En los ejemplos guiados suponemos que el HTML contiene exactamente los elementos mostrados. En código reutilizable, o cuando un elemento pueda no existir, debes comprobar el resultado de querySelector() antes de usarlo.

5. Leer y modificar contenido

5.1. Leer texto con textContent

La propiedad textContent permite leer el texto de un elemento.

HTML:

<h1 id="titulo">Página de pruebas</h1>

JavaScript:

const titulo = document.querySelector("#titulo");

console.log(titulo.textContent); // Página de pruebas

También podemos guardar ese texto en una variable:

const textoTitulo = titulo.textContent;
console.log("El título actual es:", textoTitulo);

5.2. Modificar texto con textContent

textContent también permite cambiar el texto de un elemento.

HTML:

<h1 id="titulo">Título original</h1>

JavaScript:

const titulo = document.querySelector("#titulo");

titulo.textContent = "Título modificado desde JavaScript";

Resultado en la página:

Título modificado desde JavaScript

textContent trata el contenido como texto. Si escribes etiquetas HTML, no las interpreta como etiquetas.

titulo.textContent = "<strong>Texto importante</strong>";

En la página se verá literalmente:

<strong>Texto importante</strong>

5.3. Modificar contenido HTML con innerHTML

La propiedad innerHTML permite leer o modificar el contenido HTML interno de un elemento.

HTML:

<div id="contenedor"></div>

JavaScript:

const contenedor = document.querySelector("#contenedor");

contenedor.innerHTML = "<p>Texto generado desde JavaScript</p>";

En este caso el navegador sí interpreta la etiqueta <p>.

Diferencia entre textContent e innerHTML

Propiedad Interpreta HTML Uso recomendado
textContent No Cambiar texto simple
innerHTML Sí Insertar contenido HTML controlado

Regla práctica

  • Usa textContent para textos.
  • Usa innerHTML únicamente cuando necesites crear estructura HTML a partir de una cadena controlada.
  • Cuando sea posible, crea los elementos con createElement() y asigna sus textos mediante textContent.

Precaución con innerHTML

No uses innerHTML para insertar contenido procedente del usuario o de una fuente no confiable, salvo que haya sido tratado y sanitizado específicamente.

Por ejemplo, esto puede ser peligroso en una aplicación real:

const textoUsuario = prompt("Escribe un comentario");
contenedor.innerHTML = textoUsuario;

Para texto introducido por el usuario, es más seguro usar textContent. Esto ayuda a evitar vulnerabilidades de inyección de código, como XSS.

6. Atributos, clases y estilos

6.1. Leer y modificar atributos

Los elementos HTML tienen atributos, como src, href, alt, id, class, title, etc.

Ejemplo:

<img id="logo" src="img/logo.png" alt="Logo del sitio">
<a id="enlace" href="https://example.com">Ir al sitio</a>

Podemos leer atributos con getAttribute(). Este método devuelve el valor del atributo como texto y, si el atributo no existe, devuelve null:

const logo = document.querySelector("#logo");

console.log(logo.getAttribute("src"));
console.log(logo.getAttribute("alt"));

const tituloImagen = logo.getAttribute("title");

if (tituloImagen === null) {
  console.log("La imagen no tiene atributo title");
}

Podemos modificarlos con setAttribute():

logo.setAttribute("src", "img/logo-nuevo.png");
logo.setAttribute("alt", "Nuevo logo del sitio");

Otro ejemplo con un enlace:

const enlace = document.querySelector("#enlace");

enlace.setAttribute("href", "https://developer.mozilla.org/");
enlace.textContent = "Ir a documentación";

6.2. Trabajar con clases CSS usando classList

Una forma muy habitual de cambiar el aspecto de un elemento desde JavaScript es añadir o quitar clases CSS.

HTML:

<p id="aviso">Mensaje importante</p>

CSS:

.destacado {
  color: white;
  background-color: darkred;
  padding: 10px;
}

.oculto {
  display: none;
}

JavaScript:

const aviso = document.querySelector("#aviso");

aviso.classList.add("destacado");

Métodos frecuentes de classList:

Método Qué hace
classList.add("clase") Añade una clase
classList.remove("clase") Quita una clase
classList.toggle("clase") Añade la clase si no está, y la quita si ya está
classList.contains("clase") Comprueba si el elemento tiene esa clase

Ejemplo:

const aviso = document.querySelector("#aviso");

aviso.classList.add("destacado");
console.log(aviso.classList.contains("destacado")); // true

aviso.classList.remove("destacado");

const estaOculto = aviso.classList.toggle("oculto");
console.log(estaOculto);

classList.toggle() devuelve true si la clase ha quedado añadida y false si ha quedado eliminada.

En general, es mejor cambiar clases CSS que modificar estilos directamente desde JavaScript.

6.3. Modificar estilos directamente

También se puede modificar la propiedad style de un elemento.

const titulo = document.querySelector("#titulo");

titulo.style.color = "blue";
titulo.style.backgroundColor = "lightyellow";
titulo.style.padding = "10px";

Observa que en JavaScript las propiedades CSS con guion se escriben en formato camelCase:

CSS JavaScript
background-color backgroundColor
font-size fontSize
border-radius borderRadius

Aun así, para proyectos reales es más limpio definir estilos en CSS y activar o desactivar clases con classList.

7. Crear e insertar elementos

7.1. Crear elementos con createElement()

JavaScript puede crear elementos HTML nuevos.

const parrafo = document.createElement("p");
parrafo.textContent = "Este párrafo se ha creado desde JavaScript.";

En este momento el párrafo existe en memoria, pero todavía no está dentro de la página.

Para verlo en el documento, hay que insertarlo en algún elemento del DOM.

7.2. Añadir elementos con append()

HTML:

<div id="contenedor"></div>

JavaScript:

const contenedor = document.querySelector("#contenedor");

const parrafo = document.createElement("p");
parrafo.textContent = "Párrafo añadido al contenedor.";

contenedor.append(parrafo);

También podemos crear una lista completa:

const lista = document.createElement("ul");

const elemento1 = document.createElement("li");
elemento1.textContent = "Primer elemento";

const elemento2 = document.createElement("li");
elemento2.textContent = "Segundo elemento";

lista.append(elemento1, elemento2);

const contenedor = document.querySelector("#contenedor");
contenedor.append(lista);

append() puede recibir uno o varios nodos en una misma llamada.

8. Generar contenido desde datos

8.1. Crear contenido a partir de un array

Podemos combinar lo aprendido sobre arrays con DOM.

HTML:

<ul id="lista-tareas"></ul>

JavaScript:

const tareas = ["Estudiar DOM", "Practicar JavaScript", "Repasar arrays"];
const listaTareas = document.querySelector("#lista-tareas");

tareas.forEach((tarea) => {
  const li = document.createElement("li");
  li.textContent = tarea;
  listaTareas.append(li);
});

Resultado:

<ul id="lista-tareas">
  <li>Estudiar DOM</li>
  <li>Practicar JavaScript</li>
  <li>Repasar arrays</li>
</ul>

8.2. Crear contenido a partir de un array de objetos

También podemos generar contenido usando arrays de objetos.

HTML:

<section id="catalogo"></section>

JavaScript:

const productos = [
  { nombre: "Teclado", precio: 25 },
  { nombre: "Ratón", precio: 12 },
  { nombre: "Monitor", precio: 160 }
];

const catalogo = document.querySelector("#catalogo");

productos.forEach((producto) => {
  const articulo = document.createElement("article");

  const titulo = document.createElement("h2");
  titulo.textContent = producto.nombre;

  const precio = document.createElement("p");
  precio.textContent = `Precio: ${producto.precio} €`;

  articulo.append(titulo, precio);
  catalogo.append(articulo);
});

Este patrón será muy importante en los siguientes bloques, especialmente cuando trabajemos con datos externos.

9. Vaciar y eliminar contenido

9.1. Vaciar un contenedor antes de volver a pintar

A veces necesitamos limpiar un contenedor antes de volver a generar su contenido.

HTML:

<ul id="lista"></ul>

JavaScript:

const lista = document.querySelector("#lista");

lista.innerHTML = "";

Usar innerHTML = "" para vaciar un contenedor es seguro porque no se está insertando contenido externo.

También podemos vaciarlo sin utilizar una cadena HTML:

lista.replaceChildren();

replaceChildren() elimina todos los elementos hijos del contenedor.

Después de vaciarlo, podemos volver a insertar elementos.

const nombres = ["Ana", "Luis", "María"];

lista.innerHTML = "";

nombres.forEach((nombre) => {
  const li = document.createElement("li");
  li.textContent = nombre;
  lista.append(li);
});

9.2. Eliminar elementos del DOM

Podemos eliminar un elemento con .remove().

HTML:

<p id="mensaje">Este mensaje se eliminará.</p>

JavaScript:

const mensaje = document.querySelector("#mensaje");

mensaje.remove();

También se puede eliminar un elemento seleccionado dentro de una lista:

const primerElemento = document.querySelector("li");

if (primerElemento !== null) {
  primerElemento.remove();
}

10. Comprobar si un elemento existe

Si querySelector() no encuentra ningún elemento, devuelve null.

const caja = document.querySelector("#caja");

console.log(caja); // null si no existe

Antes de modificar un elemento, conviene comprobar que existe:

const caja = document.querySelector("#caja");

if (caja !== null) {
  caja.textContent = "Caja encontrada";
} else {
  console.log("No existe ningún elemento con id caja");
}

Esto evita errores como:

Cannot set properties of null

11. Ejemplo completo guiado

HTML

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Ejemplo DOM</title>
  <link rel="stylesheet" href="estilos.css">
</head>
<body>
  <h1 id="titulo">Catálogo</h1>
  <section id="catalogo"></section>

  <script src="script.js"></script>
</body>
</html>

CSS

.producto {
  border: 1px solid #ccc;
  padding: 10px;
  margin-bottom: 10px;
}

.caro {
  background-color: #ffe5e5;
}

JavaScript

const productos = [
  { nombre: "Teclado", precio: 25 },
  { nombre: "Ratón", precio: 12 },
  { nombre: "Monitor", precio: 160 }
];

const titulo = document.querySelector("#titulo");
const catalogo = document.querySelector("#catalogo");

titulo.textContent = "Catálogo de productos";

function crearElementoProducto(producto) {
  const articulo = document.createElement("article");
  articulo.classList.add("producto");

  if (producto.precio >= 100) {
    articulo.classList.add("caro");
  }

  const nombre = document.createElement("h2");
  nombre.textContent = producto.nombre;

  const precio = document.createElement("p");
  precio.textContent = `Precio: ${producto.precio} €`;

  articulo.append(nombre, precio);

  return articulo;
}

productos.forEach((producto) => {
  catalogo.append(crearElementoProducto(producto));
});

Este ejemplo separa la creación de cada producto en una función reutilizable y combina:

  • selección de elementos;
  • modificación de texto;
  • creación de elementos;
  • uso de clases CSS;
  • recorrido de arrays de objetos;
  • inserción dinámica en el DOM.

12. Buenas prácticas al manipular el DOM

  • Usa nombres de variables claros.
  • Comprueba que el elemento existe si puede no estar en la página.
  • Usa textContent para texto normal.
  • Usa innerHTML solo cuando necesites insertar HTML controlado.
  • Prefiere classList para cambiar estilos.
  • Mantén separado el HTML, CSS y JavaScript.
  • No mezcles demasiadas operaciones distintas en una misma función.
  • Crea funciones para renderizar contenido cuando el código empiece a crecer.

Ejemplo:

function crearElementoProducto(producto) {
  const articulo = document.createElement("article");
  articulo.classList.add("producto");

  const nombre = document.createElement("h2");
  nombre.textContent = producto.nombre;

  const precio = document.createElement("p");
  precio.textContent = `Precio: ${producto.precio} €`;

  articulo.append(nombre);
  articulo.append(precio);

  return articulo;
}

Uso:

const catalogo = document.querySelector("#catalogo");

productos.forEach((producto) => {
  const elementoProducto = crearElementoProducto(producto);
  catalogo.append(elementoProducto);
});

13. Errores frecuentes

Error 1. Seleccionar un elemento que no existe

const titulo = document.querySelector("#tituloPrincipal");
titulo.textContent = "Nuevo título";

Si no existe ningún elemento con id="tituloPrincipal", titulo será null.

Corrección:

const titulo = document.querySelector("#tituloPrincipal");

if (titulo !== null) {
  titulo.textContent = "Nuevo título";
}

Error 2. Confundir querySelector y querySelectorAll

const parrafos = document.querySelectorAll("p");
parrafos.textContent = "Texto nuevo";

parrafos contiene varios elementos, no uno solo.

Corrección:

const parrafos = document.querySelectorAll("p");

parrafos.forEach((parrafo) => {
  parrafo.textContent = "Texto nuevo";
});

Error 3. Usar mal el selector

const titulo = document.querySelector("titulo");

Esto busca una etiqueta <titulo>, no un elemento con id="titulo".

Corrección:

const titulo = document.querySelector("#titulo");

Error 4. Crear un elemento pero no insertarlo

const parrafo = document.createElement("p");
parrafo.textContent = "Hola";

El párrafo se ha creado, pero no aparece en la página porque no se ha añadido al DOM.

Corrección:

const contenedor = document.querySelector("#contenedor");

const parrafo = document.createElement("p");
parrafo.textContent = "Hola";

contenedor.append(parrafo);

Error 5. Usar innerHTML para texto de usuario

const comentario = prompt("Escribe un comentario");
contenedor.innerHTML = comentario;

Para texto de usuario, mejor:

const comentario = prompt("Escribe un comentario");
contenedor.textContent = comentario;

16. Resumen final

  • El DOM es la representación de una página HTML como una estructura de objetos.
  • document representa la página cargada en el navegador.
  • querySelector() selecciona el primer elemento que coincide con un selector.
  • querySelectorAll() selecciona todos los elementos que coinciden con un selector.
  • textContent permite leer o cambiar texto.
  • innerHTML permite insertar HTML, pero debe usarse con cuidado.
  • getAttribute() y setAttribute() permiten leer y modificar atributos.
  • classList permite añadir, quitar o alternar clases CSS.
  • createElement() permite crear elementos nuevos.
  • append() permite insertar elementos en la página.
  • .remove() permite eliminar elementos.
  • Podemos generar contenido dinámico a partir de arrays y objetos.
  • Este bloque prepara el camino para el BOM, los eventos, los formularios y las aplicaciones dinámicas.
