Práctica global · UD2 B1

UD2_B1. Práctica global: objetos, arrays y JSON

Desarrollo de la lógica interna de un catálogo de productos mediante objetos, arrays de objetos, funciones y JSON, sin utilizar todavía el DOM.

1. Objetivo de la práctica

En esta práctica vas a desarrollar un pequeño programa en JavaScript para gestionar un catálogo de productos usando objetos, arrays de objetos y JSON.

La práctica sirve para cerrar el bloque UD2_B1: Objetos, estructuras de datos y JavaScript avanzado sin DOM.

No se trabajará todavía con HTML, DOM, eventos, formularios ni asincronía. Todo el resultado se mostrará por consola.

2. Qué contenidos vas a practicar

En esta práctica aplicarás:

  • objetos literales y acceso a propiedades;
  • arrays de objetos;
  • funciones que reciben arrays y objetos;
  • forEach;
  • map;
  • filter;
  • reduce;
  • find;
  • some;
  • every;
  • sort() con función de comparación;
  • destructuring;
  • spread;
  • JSON.stringify();
  • JSON.parse();
  • comentarios y organización del código;
  • pruebas por consola.

3. Enunciado general

Vas a crear un programa llamado Gestor de productos de una tienda.

El programa debe partir de un array de productos. Cada producto será un objeto con información básica:

  • identificador;
  • nombre;
  • categoría;
  • precio;
  • stock;
  • si está o no en oferta.

A partir de esos datos, deberás crear funciones para consultar, transformar, filtrar, ordenar, calcular totales y exportar información en formato JSON.

6. Parte 1: exploración inicial de datos

Ejercicio 1. Mostrar todos los productos

Crea una función llamada mostrarProductos(productos) que reciba el array de productos y muestre por consola una línea por cada producto.

El formato debe ser similar a este:

1 - Portátil Lenovo IdeaPad | informatica | 699.99 € | stock: 8

Requisitos:

  • usa forEach;
  • usa destructuring dentro del callback;
  • no modifiques el array original.

Ejemplo orientativo:

function mostrarProductos(productos) {
  productos.forEach((producto) => {
    const { id, nombre, categoria, precio, stock } = producto;
    console.log(`${id} - ${nombre} | ${categoria} | ${precio} € | stock: ${stock}`);
  });
}

Ejercicio 2. Mostrar solo nombre y precio

Crea una función llamada mostrarNombreYPrecio(productos) que muestre únicamente el nombre y el precio de cada producto.

Ejemplo de salida:

Portátil Lenovo IdeaPad: 699.99 €
Ratón Logitech M185: 12.99 €

Requisitos:

  • usa forEach;
  • usa destructuring;
  • el código debe ser claro y estar bien indentado.

Ejercicio 5. Aplicar descuento a productos en oferta

Crea una función llamada aplicarDescuento(productos, porcentaje).

La función debe devolver un nuevo array en el que:

  • los productos en oferta tengan el precio rebajado según el porcentaje indicado;
  • los productos que no están en oferta mantengan su precio original;
  • cada producto conserve el resto de propiedades.

Ejemplo:

const productosConDescuento = aplicarDescuento(productos, 10);
console.log(productosConDescuento);

Para esta práctica se supondrá que porcentaje es un número válido entre 0 y 100.

Requisitos:

  • usa map;
  • usa spread ... para copiar cada producto;
  • no modifiques el array productos original;
  • redondea el precio final a dos decimales usando toFixed(2) y Number().

Ejemplo orientativo:

const precioRebajado = Number((precio * (1 - porcentaje / 100)).toFixed(2));

8. Parte 3: filtrado de datos con filter

Ejercicio 6. Filtrar productos con stock

Crea una función llamada filtrarConStock(productos) que devuelva un nuevo array solo con los productos cuyo stock sea mayor que 0.

Requisitos:

  • usa filter;
  • no modifiques el array original.

Ejercicio 7. Filtrar productos por categoría

Crea una función llamada filtrarPorCategoria(productos, categoriaBuscada).

La función debe devolver los productos cuya categoría coincida con categoriaBuscada.

Ejemplo:

const perifericos = filtrarPorCategoria(productos, "perifericos");
console.log(perifericos);

Requisitos:

  • usa filter;
  • elimina posibles espacios al principio y al final de la categoría recibida;
  • la comparación debe funcionar aunque la categoría se escriba con mayúsculas o minúsculas distintas;
  • usa trim() y toLowerCase().

Ejemplo orientativo:

function filtrarPorCategoria(productos, categoriaBuscada) {
  const categoriaNormalizada = categoriaBuscada.trim().toLowerCase();

  return productos.filter((producto) => {
    return producto.categoria.toLowerCase() === categoriaNormalizada;
  });
}

En los datos iniciales las categorías se escriben como "informatica", "perifericos" y "mobiliario", sin tilde. Usa esos mismos valores al realizar las pruebas.

Ejercicio 8. Filtrar productos por precio máximo

Debe devolver un nuevo array con los productos cuyo precio sea menor o igual que precioMaximo.

Ejemplo:

const baratos = filtrarPorPrecioMaximo(productos, 50);
console.log(baratos);

10. Parte 5: cálculos con reduce

Ejercicio 12. Calcular valor total del inventario

Crea una función llamada calcularValorInventario(productos).

Debe calcular el valor total del inventario teniendo en cuenta precio y stock:

precio * stock

Ejemplo:

const totalInventario = calcularValorInventario(productos);
console.log("Valor total del inventario:", totalInventario);

Requisitos:

  • usa reduce;
  • el valor inicial del acumulador debe ser 0;
  • redondea el resultado final a dos decimales;
  • devuelve un número, no una cadena.

Puedes hacerlo así:

return Number(total.toFixed(2));

Ejercicio 13. Contar unidades totales en stock

Crea una función llamada contarUnidadesEnStock(productos).

Debe devolver la suma total de unidades disponibles en stock.

Requisitos:

  • usa reduce;
  • no cuentes productos, cuenta unidades.

Ejercicio 14. Calcular precio medio

Crea una función llamada calcularPrecioMedio(productos).

Debe devolver el precio medio de los productos.

Requisitos:

  • usa reduce para sumar los precios;
  • divide entre productos.length;
  • redondea a dos decimales;
  • devuelve un número, no una cadena;
  • controla que el array no esté vacío.

Si el array está vacío, la función debe devolver 0.

Ejemplo de redondeo:

return Number(precioMedio.toFixed(2));

11. Parte 6: ordenación con sort

Ejercicio 15. Ordenar productos por precio ascendente

Crea una función llamada ordenarPorPrecio(productos).

Debe devolver un nuevo array ordenado de menor a mayor precio.

Requisitos:

  • usa sort con función de comparación;
  • usa spread para no modificar el array original.

Ejemplo orientativo:

function ordenarPorPrecio(productos) {
  return [...productos].sort((a, b) => a.precio - b.precio);
}

Ejercicio 16. Ordenar productos por stock descendente

Crea una función llamada ordenarPorStockDescendente(productos).

Debe devolver un nuevo array ordenado desde el producto con más stock hasta el producto con menos stock.

Requisitos:

  • usa sort;
  • no modifiques el array original.

15. Parte 10: corrige los errores

Analiza los siguientes fragmentos y corrige los errores.

Error 1. map sin return

const nombres = productos.map((producto) => {
  producto.nombre;
});

console.log(nombres);

Explica por qué el resultado no es el esperado.

Error 2. sort modificando el array original

const ordenados = productos.sort((a, b) => a.precio - b.precio);
console.log(productos);
console.log(ordenados);

Corrige el código para conservar el array original sin cambios.

Error 3. Comparación incorrecta en filter

const informatica = productos.filter((producto) => producto.categoria = "informatica");
console.log(informatica);

Explica el error y corrige el operador.

Error 4. reduce sin valor inicial

const total = productos.reduce((acumulador, producto) => {
  return acumulador + producto.precio;
});

Corrige el código añadiendo un valor inicial adecuado.

Sin valor inicial, el primer elemento del array se usa como acumulador. En este caso sería un objeto producto, no el número 0, por lo que la suma no se realizaría correctamente.

20. Resumen final

En esta práctica has usado JavaScript para trabajar con una colección de datos realista. El objetivo no es solo resolver ejercicios sueltos, sino empezar a pensar como si estuvieras preparando la lógica interna de una pequeña aplicación.

Los datos todavía se muestran por consola, pero la forma de trabajar con objetos, arrays y JSON será la base para los siguientes bloques, donde esos datos se mostrarán y modificarán desde una página web usando DOM, eventos y comunicación asíncrona.
