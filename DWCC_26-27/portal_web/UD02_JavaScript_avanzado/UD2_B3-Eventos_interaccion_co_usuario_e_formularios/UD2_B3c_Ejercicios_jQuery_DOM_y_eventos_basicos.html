UD2_B3c. Boletín de ejercicios: jQuery, DOM y eventos básicos

1. Objetivo y preparación del proyecto

1.1. Objetivo del boletín

Este boletín sirve para practicar el uso básico de jQuery como biblioteca de apoyo para manipular el DOM y gestionar eventos.

jQuery no sustituye al aprendizaje del DOM nativo ni de JavaScript moderno. En este bloque se utiliza como complemento para:

  • reconocer código existente en proyectos heredados;
  • comparar soluciones hechas con JavaScript estándar y con jQuery;
  • simplificar operaciones frecuentes sobre elementos HTML;
  • reforzar selectores, eventos y manipulación del DOM;
  • comprender la diferencia entre un elemento DOM y un objeto jQuery;
  • practicar eventos directos y delegados;
  • utilizar correctamente .attr(), .prop(), .text(), .html() y .val().

En estos ejercicios no trabajaremos AJAX de jQuery como contenido principal. Para la comunicación asíncrona seguiremos priorizando fetch(), promesas y async/await.

1.2. Estructura de archivos

Crea una carpeta llamada:

UD2_B3c_Ejercicios_jQuery

Dentro de ella crea:

UD2_B3c_Ejercicios_jQuery/
├── index.html
├── styles.css
└── script.js

1.3. Cargar jQuery

En index.html, carga primero jQuery y después tu archivo:

<script src="https://code.jquery.com/jquery-4.0.0.min.js"></script>
<script src="script.js"></script>

En proyectos heredados puedes encontrar jQuery 3.7.1 o versiones anteriores. No actualices automáticamente una aplicación existente sin revisar su compatibilidad.

Si trabajas sin conexión, usa una copia local:

<script src="js/jquery-4.0.0.min.js"></script>
<script src="js/script.js"></script>

1.4. Comprobar la versión cargada

En la consola:

console.log($.fn.jquery);

Resultado esperado:

4.0.0

Si aparece $ is not defined, revisa el orden de los scripts.
