UD2_B2. Ejercicios: DOM básico

Selección, lectura, modificación, creación y renderizado de elementos del DOM mediante JavaScript, sin eventos ni formularios.

1. Objetivo de la hoja

En esta hoja vas a practicar el uso básico del DOM (Document Object Model) en JavaScript.

El objetivo no es todavía crear aplicaciones completas con eventos o formularios, sino aprender a:

  • seleccionar elementos de una página HTML;
  • leer y modificar texto;
  • modificar contenido HTML de forma controlada;
  • cambiar atributos;
  • añadir y quitar clases CSS;
  • crear nuevos elementos desde JavaScript;
  • insertar y eliminar elementos del documento;
  • recorrer varios elementos seleccionados con querySelectorAll.

En esta hoja no vamos a trabajar todavía eventos, addEventListener, formularios, fetch ni comunicación asíncrona. Eso se verá en bloques posteriores de la UD2.

2. Preparación común para todos los ejercicios

Crea una carpeta llamada:

UD2_B2_DOM_basico

Dentro crea estos archivos:

UD2_B2_DOM_basico/
├── index.html
├── css/
│   └── estilos.css
├── img/
│   ├── logo1.png
│   └── logo2.png
└── js/
    └── script.js

Usa esta plantilla inicial en index.html:

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>UD2_B2 - DOM básico</title>
  <link rel="stylesheet" href="css/estilos.css">
</head>
<body>
  <h1 id="titulo-principal">Práctica de DOM básico</h1>

  <p id="mensaje">Este texto será modificado con JavaScript.</p>

  <section id="contenedor">
    <h2>Lista inicial</h2>
    <ul id="lista">
      <li class="item">HTML</li>
      <li class="item">CSS</li>
      <li class="item">JavaScript</li>
    </ul>
  </section>

  <section id="zona-dinamica">
    <h2>Zona dinámica</h2>
  </section>

  <script src="js/script.js"></script>
</body>
</html>

Usa este CSS inicial en css/estilos.css:

.destacado {
  font-weight: bold;
  background-color: #fff3cd;
  padding: 4px;
}

.correcto {
  color: green;
}

.error {
  color: red;
}

.tarjeta {
  border: 1px solid #ccc;
  padding: 10px;
  margin: 8px 0;
  border-radius: 6px;
}

Realiza cada ejercicio en js/script.js.

Importante: trabaja un ejercicio cada vez

Los ejercicios reutilizan los mismos elementos HTML y algunos emplean nombres de variables similares. No dejes todos los fragmentos activos simultáneamente en script.js.

Para cada ejercicio:

  1. comenta temporalmente el código anterior;
  2. escribe y ejecuta el ejercicio actual;
  3. recarga la página para recuperar el HTML inicial;
  4. comprueba el resultado antes de continuar.

También puedes organizar cada ejercicio dentro de una función:

function ejercicio1() {
  const titulo = document.querySelector("#titulo-principal");
  console.log(titulo);
}

ejercicio1();

Deja activa únicamente la llamada del ejercicio que estés comprobando.

Ejecución: estos ejercicios deben abrirse desde index.html en el navegador. Si ejecutas script.js directamente con Node.js, aparecerá un error similar a:

ReferenceError: document is not defined

Parte 1. Seleccionar elementos del DOM

console.log(lista.textContent);

Después responde en un comentario de código:

// ¿Qué diferencia ves entre mostrar lista y mostrar lista.textContent?

Ejercicio 3. Seleccionar varios elementos

  1. Selecciona todos los elementos con clase item usando document.querySelectorAll().
  2. Muestra por consola el resultado.
  3. Muestra cuántos elementos se han seleccionado usando .length.

console.log(items);

// ¿El resultado es un array normal o una NodeList?

La NodeList obtenida con querySelectorAll() es estática y puede recorrerse con for o forEach.

Parte 3. Modificar HTML de forma controlada

Ejercicio 7. Usar innerHTML

  1. Selecciona la sección zona-dinamica.
  2. Añade dentro de ella un párrafo con el texto:
Este párrafo se ha creado usando innerHTML.

Ejemplo:

  <h2>Zona dinámica</h2>

Ejercicio 8. Diferenciar textContent e innerHTML

Prueba este código:

const zona = document.querySelector("#zona-dinamica");

zona.textContent = "<strong>Texto importante</strong>";

Después prueba este otro:

zona.innerHTML = "<strong>Texto importante</strong>";

Contesta en comentarios:

// ¿Qué ocurre con textContent?
// ¿Qué ocurre con innerHTML?
// ¿Cuál de los dos interpreta etiquetas HTML?

Ejercicio 9. Precaución con innerHTML

Escribe un comentario explicando por qué no debemos insertar con innerHTML contenido que venga directamente de un usuario sin comprobarlo antes.

No hace falta profundizar todavía en ataques concretos. La idea básica es:

// innerHTML interpreta etiquetas HTML.
// No debe usarse con contenido del usuario o de fuentes no confiables
// salvo que se haya tratado y sanitizado específicamente.
// Para mostrar texto introducido por el usuario, es más seguro usar textContent.

Esta precaución ayuda a evitar vulnerabilidades de inyección de código, como XSS.

Parte 4. Modificar atributos

Ejercicio 10. Crear una imagen en HTML y modificarla

Añade esta imagen dentro del body de index.html, antes del script:

<img id="logo" src="img/logo1.png" alt="Logo inicial">

Guarda dos imágenes cualesquiera dentro de la carpeta img y renómbralas como:

logo1.png
logo2.png

No importa su contenido: solo se usarán para comprobar el cambio del atributo src.

En JavaScript:

  1. Selecciona la imagen.
  2. Cambia su atributo src a "img/logo2.png".
  3. Cambia su atributo alt a "Logo modificado desde JavaScript".
  4. Muestra por consola los nuevos valores.
const logo = document.querySelector("#logo");

logo.setAttribute("src", "img/logo2.png");
logo.setAttribute("alt", "Logo modificado desde JavaScript");

console.log(logo.getAttribute("src"));
console.log(logo.getAttribute("alt"));

Parte 6. Crear elementos desde JavaScript

Ejercicio 15. Crear un párrafo

  1. Crea un elemento p con document.createElement.
  2. Asígnale texto con .textContent.
  3. Insértalo dentro de zona-dinamica usando append.
const zona = document.querySelector("#zona-dinamica");

const parrafo = document.createElement("p");
parrafo.textContent = "Párrafo creado con createElement.";

zona.append(parrafo);

Ejercicio 16. Crear un elemento de lista

  1. Selecciona la lista con id lista.
  2. Crea un nuevo elemento li.
  3. Asígnale el texto "DOM".
  4. Añade la clase item.
  5. Insértalo al final de la lista.
const lista = document.querySelector("#lista");

const nuevoItem = document.createElement("li");
nuevoItem.textContent = "DOM";
nuevoItem.classList.add("item");

lista.append(nuevoItem);

Parte 8. Renderizar datos en el DOM

Ejercicio 20. Renderizar un array de textos

Dado este array:

const lenguajes = ["HTML", "CSS", "JavaScript", "PHP"];
  1. Selecciona la lista #lista.
  2. Vacía su contenido.
  3. Recorre el array con un for.
  4. Por cada lenguaje, crea un li.
  5. Inserta cada li en la lista.

Pista:

const lista = document.querySelector("#lista");
const lenguajes = ["HTML", "CSS", "JavaScript", "PHP"];

lista.textContent = "";

for (let i = 0; i < lenguajes.length; i++) {
  const li = document.createElement("li");
  li.textContent = lenguajes[i];
  li.classList.add("item");
  lista.append(li);
}

Parte 10. Mini-entrega final

Ejercicio 26. Página dinámica de resumen de módulos

Crea una pequeña página dinámica usando DOM.

Datos de partida

En script.js, usa este array:

const modulosResumen = [
  { nombre: "DWCC", descripcion: "Desenvolvemento web en contorno cliente", horas: 219 },
  { nombre: "DWES", descripcion: "Desenvolvemento web en contorno servidor", horas: 157 },
  { nombre: "DIW", descripcion: "Deseño de interfaces web", horas: 107 }
];

Requisitos

Tu programa debe:

  1. Seleccionar zona-dinamica.
  2. Vaciar su contenido inicial.
  3. Crear un título h2 con el texto "Módulos del ciclo".
  4. Recorrer modulosResumen y crear una tarjeta por cada módulo.
  5. Cada tarjeta debe incluir:
    • un título h3 con el nombre del módulo;
    • un párrafo con la descripción;
    • un párrafo con las horas.
  6. Añadir la clase tarjeta a cada tarjeta.
  7. Al final, crear un párrafo con el total de módulos.
  8. No usar eventos.
  9. No usar innerHTML para crear las tarjetas.

Entrega

Entrega:

  • index.html;
  • css/estilos.css;
  • js/script.js;
  • una captura del navegador;
  • una captura de la consola sin errores.
