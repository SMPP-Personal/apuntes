UD01. Práctica global integradora: JavaScript básico

¡Unidad 1 completada! Esta práctica reúne todo el JavaScript básico trabajado hasta ahora y sirve como puente festivo hacia la UD2.

Fin de UD01 Práctica global JavaScript básico Rumbo a UD2

1. Contexto de la práctica

Esta práctica sirve para cerrar la UD01: JavaScript básico en contorno cliente.

🎉 Momento celebración: esta práctica marca el cierre de la UD01. No es solo una entrega: es la demostración de que ya puedes combinar piezas de JavaScript para resolver un problema completo.

El objetivo es construir un pequeño programa en JavaScript que permita trabajar con una lista de notas de un grupo. La práctica integra contenidos vistos durante la unidad:

  • sintaxis básica de JavaScript;
  • variables, constantes y tipos de datos;
  • operadores;
  • condicionales;
  • bucles;
  • funciones;
  • arrays básicos;
  • métodos avanzados de arrays;
  • objetos predefinidos (String, Number, Math, Date);
  • uso básico de JSON.

No se trabajará todavía con DOM, eventos, objetos propios ni asincronía. La salida del programa se mostrará por consola.

2. Relación con la unidad

Esta práctica consolida especialmente los contenidos de:

  • B02: sintaxis, variables y tipos;
  • B03: operadores, condicionales y bucles;
  • B04: funciones básicas;
  • B04b: funciones anónimas, funciones flecha y callbacks básicos;
  • B05A: arrays básicos;
  • B05B: métodos avanzados de arrays;
  • B05C: objetos predefinidos.

Según la programación del módulo, la UD01 trabaja la base de JavaScript: sintaxis, tipos de datos, operadores, estructuras de control, funciones y arrays básicos. Esta práctica actúa como evidencia final de integración antes de avanzar hacia JavaScript avanzado.

3. Archivos que debes crear

Crea una carpeta llamada:

UD01_practica_global

Dentro de ella crea, como mínimo, estos archivos:

index.html
script.js

El archivo index.html solo necesita cargar el JavaScript y permitir abrir la consola del navegador.

Ejemplo mínimo:

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Práctica global UD01</title>
</head>
<body>
  <h1>Práctica global UD01: JavaScript básico</h1>
  <p>Abre la consola del navegador para ver los resultados.</p>

  <script src="script.js"></script>
</body>
</html>

Todo el trabajo principal se realizará en script.js.

4. Enunciado general

Vas a crear un programa para analizar las notas de un grupo.

El programa partirá de un array de notas introducidas inicialmente como texto. Deberás limpiar los datos, convertirlos a números, descartar valores incorrectos, calcular estadísticas y mostrar un resumen final.

Array inicial:

const notasTexto = [" 7.5 ", "4", "9", "cinco", "6.25", "", "10", "3.5", "8"];

El programa deberá:

  1. Limpiar los datos de texto.
  2. Convertir las notas válidas a número.
  3. Detectar y descartar los valores no válidos.
  4. Calcular estadísticas del grupo.
  5. Generar mensajes de resultado.
  6. Crear una versión JSON del resumen final.
  7. Mostrar por consola todos los resultados de forma ordenada.

5. Requisitos obligatorios

5.1. Variables, constantes y tipos

Declara las constantes y variables necesarias para el programa.

Como mínimo deberás usar:

  • const para valores que no se reasignan;
  • let para acumuladores o variables que cambian;
  • cadenas de texto;
  • números;
  • booleanos;
  • arrays;
  • valores NaN o comprobaciones relacionadas.

5.2. Limpieza y conversión de datos

Crea una función llamada convertirNota.

Debe recibir una nota en formato texto y devolver:

  • un número, si la conversión es válida;
  • null, si la nota no es válida.

Ejemplo de uso esperado:

console.log(convertirNota(" 7.5 "));  // 7.5
console.log(convertirNota("cinco"));  // null
console.log(convertirNota(""));       // null

Pistas:

  • Usa .trim() para eliminar espacios.
  • Usa Number() o parseFloat().
  • Usa Number.isNaN() para comprobar conversiones incorrectas.
  • Una nota válida debe estar entre 0 y 10.

5.3. Crear el array de notas válidas

A partir de notasTexto, crea un nuevo array llamado notas.

Puedes hacerlo con un bucle for o con métodos avanzados de array.

El resultado esperado debe ser similar a:

[7.5, 4, 9, 6.25, 10, 3.5, 8]

No deben aparecer:

  • "cinco";
  • "";
  • valores null;
  • valores NaN.

5.4. Mostrar las notas válidas

Usa forEach para mostrar cada nota válida con su posición:

Nota 1: 7.5
Nota 2: 4
Nota 3: 9
...

Recuerda que el índice del array empieza en 0, pero para mostrar al usuario puede ser más claro empezar en 1.

5.5. Calcular la suma

Crea una función llamada calcularSuma.

Debe recibir un array de números y devolver la suma total.

Puedes resolverla con un bucle for o con reduce.

Ejemplo:

const suma = calcularSuma(notas);
console.log("Suma:", suma);

5.6. Calcular la media

Crea una función llamada calcularMedia.

Debe recibir un array de números y devolver la media.

Ten en cuenta:

  • si el array está vacío, no se puede calcular una media real;
  • en ese caso, devuelve null.

Ejemplo:

const media = calcularMedia(notas);
console.log("Media:", media);

Muestra la media redondeada a dos decimales usando .toFixed(2).

5.7. Contar aprobados y suspensos

Crea dos arrays:

const aprobados = ...
const suspensos = ...

Usa filter para obtener:

  • notas mayores o iguales que 5;
  • notas menores que 5.

Después muestra:

Aprobados: X
Suspensos: Y

5.8. Transformar notas en textos

Usa map para crear un array llamado resultadosTexto.

Debe convertir cada nota en un texto:

  • "Aprobado" si la nota es mayor o igual que 5;
  • "Suspenso" si la nota es menor que 5.

Ejemplo:

[ "Aprobado", "Suspenso", "Aprobado", ... ]

5.9. Buscar la nota máxima y mínima

Crea dos funciones:

function obtenerNotaMaxima(notas) {
  // ...
}

function obtenerNotaMinima(notas) {
  // ...
}

Puedes usar Math.max() y Math.min() o un bucle for.

Como todavía no hemos trabajado spread/rest como contenido principal, si usas Math.max(...notas) debes añadir un comentario indicando que ... se estudiará más adelante.

Recomendación para este nivel: resolverlo con un bucle for.

5.10. Clasificar la media del grupo

Crea una función llamada clasificarMedia.

Debe recibir una media numérica y devolver un texto:

Media Resultado
Menor que 5 "Grupo con media insuficiente"
Entre 5 y 6.99 "Grupo aprobado"
Entre 7 y 8.99 "Grupo con buen rendimiento"
Entre 9 y 10 "Grupo excelente"

Ejemplo:

const clasificacion = clasificarMedia(media);
console.log(clasificacion);

5.11. Mostrar la fecha del informe

Usa Date para mostrar la fecha actual del informe.

El formato debe ser:

Informe generado el 14/6/2026

Recuerda:

const hoy = new Date();
const dia = hoy.getDate();
const mes = hoy.getMonth() + 1;
const anyo = hoy.getFullYear();

5.12. Crear un resumen final

Crea un resumen con los datos principales.

Como todavía no hemos trabajado objetos propios como contenido principal, puedes crear el resumen de una de estas dos formas.

Opción A: usando variables separadas

const totalNotas = notas.length;
const numeroAprobados = aprobados.length;
const numeroSuspensos = suspensos.length;

Opción B: usando un objeto sencillo

Puedes usar un objeto literal básico solo como estructura de datos:

const resumen = {
  totalNotas: notas.length,
  media: Number(media.toFixed(2)),
  aprobados: aprobados.length,
  suspensos: suspensos.length,
  maxima: maxima,
  minima: minima,
  fecha: fechaInforme
};

No es necesario profundizar todavía en objetos propios. En UD2 se trabajarán con más detalle.

5.13. Convertir el resumen a JSON

Usa JSON.stringify() para convertir el resumen final a texto JSON.

Ejemplo:

const resumenJSON = JSON.stringify(resumen);
console.log(resumenJSON);

Después usa JSON.parse() para recuperar el resumen:

const resumenRecuperado = JSON.parse(resumenJSON);
console.log(resumenRecuperado);

13. Errores frecuentes que debes revisar

Antes de entregar, comprueba que no has cometido estos errores:

  • usar los textos originales para calcular la media;
  • no eliminar espacios antes de convertir;
  • no comprobar si aparece NaN;
  • aceptar notas menores que 0 o mayores que 10;
  • usar forEach esperando que devuelva un nuevo array;
  • olvidar el return en map, filter o reduce;
  • dividir entre 0 si el array de notas está vacío;
  • olvidar sumar 1 al mes obtenido con getMonth();
  • escribir un JSON manualmente en lugar de usar JSON.stringify();
  • no comprobar que JSON.parse() recupera los datos correctamente.
