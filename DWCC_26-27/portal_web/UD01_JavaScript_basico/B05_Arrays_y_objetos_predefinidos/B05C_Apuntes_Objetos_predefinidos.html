Bloque 5C. Objetos predefinidos en JavaScript

1. Qué vamos a aprender en este bloque

En los bloques anteriores ya hemos trabajado:

  • variables, tipos de datos y operadores;
  • condicionales y bucles;
  • funciones;
  • arrays básicos;
  • métodos avanzados de arrays.

En este bloque vamos a trabajar con algunos objetos predefinidos de JavaScript que nos ayudan a resolver tareas habituales:

  • trabajar con textos;
  • convertir datos a números;
  • hacer cálculos matemáticos;
  • generar números aleatorios;
  • obtener fechas y horas;
  • convertir datos a formato JSON y recuperarlos después.

Los objetos predefinidos que veremos son:

Objeto o herramienta Para qué sirve
String Trabajar con cadenas de texto
Number / conversión numérica Convertir valores y trabajar con números
Math Realizar operaciones matemáticas
Date Trabajar con fechas y horas
JSON Convertir datos a texto JSON y viceversa

Importante: en este bloque no vamos a estudiar todavía objetos propios en profundidad. Eso se verá más adelante en JavaScript avanzado. Aquí usaremos los objetos predefinidos como herramientas prácticas.

2. Qué es un objeto predefinido

JavaScript incluye objetos ya preparados que podemos usar directamente. Son como cajas de herramientas.

Un objeto puede tener:

  • propiedades: datos asociados al objeto;
  • métodos: funciones asociadas al objeto.

Por ejemplo:

const texto = "DWCC";

console.log(texto.length);        // propiedad
console.log(texto.toLowerCase()); // método

En este caso:

  • length es una propiedad;
  • toLowerCase() es un método.

La diferencia visual es importante:

texto.length        // propiedad: no lleva paréntesis
texto.toLowerCase() // método: lleva paréntesis

3. String: trabajar con texto

Las cadenas de texto tienen propiedades y métodos útiles para consultar, transformar o comprobar su contenido.

Nota Normalmente no creamos textos con new String(). En el curso trabajaremos con cadenas normales, como "Hola", y usaremos sus propiedades y métodos directamente.

const modulo = "Desarrollo Web en Entorno Cliente";

3.1. Longitud de una cadena: .length

La propiedad .length indica cuántos caracteres tiene una cadena.

const texto = "DWCC";

console.log(texto.length); // 4

También cuenta espacios:

const mensaje = "Hola clase";

console.log(mensaje.length); // 10

Se usa mucho para validar textos:

const password = "abc123";

if (password.length < 8) {
  console.log("La contraseña es demasiado corta");
}

3.2. Mayúsculas y minúsculas: .toUpperCase() y .toLowerCase()

Estos métodos devuelven una nueva cadena transformada:

const nombre = "Ana";

console.log(nombre.toUpperCase()); // "ANA"
console.log(nombre.toLowerCase()); // "ana"

La cadena original no cambia:

const ciudad = "Ourense";

const ciudadMayusculas = ciudad.toUpperCase();

console.log(ciudad);           // "Ourense"
console.log(ciudadMayusculas); // "OURENSE"

Esto es útil para comparar textos sin depender de mayúsculas y minúsculas:

const respuesta = "Si";

if (respuesta.toLowerCase() === "si") {
  console.log("Respuesta afirmativa");
}

3.3. Quitar espacios al principio y al final: .trim()

El método .trim() elimina los espacios en blanco al principio y al final de una cadena.

const entrada = "   hola DWCC   ";

const limpia = entrada.trim();

console.log(limpia); // "hola DWCC"

Es muy útil cuando se leen datos escritos por el usuario:

const usuario = "   Ana   ";

if (usuario.trim() === "Ana") {
  console.log("Usuario correcto");
}

.trim() no elimina los espacios que están en medio del texto, solo los del principio y los del final.

3.4. Comprobar si contiene un texto: .includes()

El método .includes() devuelve true si la cadena contiene el texto buscado.

const mensaje = "Bienvenido al módulo de DWCC";

console.log(mensaje.includes("DWCC")); // true
console.log(mensaje.includes("DWES")); // false

Ejemplo práctico:

const correo = "alumna@centro.gal";

if (correo.includes("@")) {
  console.log("Parece un correo válido");
} else {
  console.log("Falta la arroba");
}

3.5. Comprobar inicio y final: .startsWith() y .endsWith()

.startsWith() comprueba si una cadena empieza por un texto determinado.

const archivo = "practica.js";

console.log(archivo.startsWith("prac")); // true

.endsWith() comprueba si una cadena termina por un texto determinado.

const archivo = "practica.js";

console.log(archivo.endsWith(".js")); // true

Ejemplo:

const nombreArchivo = "ejercicio.html";

if (nombreArchivo.endsWith(".html")) {
  console.log("Es un archivo HTML");
}

3.6. Extraer una parte de un texto: .slice()

El método .slice(inicio, fin) permite obtener una parte de una cadena.

const modulo = "DWCC";

console.log(modulo.slice(0, 2)); // "DW"
console.log(modulo.slice(2));    // "CC"

El índice inicial se incluye y el índice final no se incluye.

const texto = "JavaScript";

console.log(texto.slice(0, 4)); // "Java"

4. Number y conversión numérica

En JavaScript, muchas veces los datos llegan como texto, aunque parezcan números.

Por ejemplo, si usamos prompt, el resultado siempre es una cadena:

const edadTexto = prompt("Introduce tu edad");

console.log(typeof edadTexto); // "string"

prompt() está disponible en el navegador. Si ejecutas JavaScript con Node.js, este ejemplo no funcionará directamente.

Para poder hacer operaciones matemáticas, necesitamos convertir ese texto a número.

4.1. Convertir a número: Number()

La función Number() intenta convertir un valor a número.

console.log(Number("123"));   // 123
console.log(Number("3.14"));  // 3.14
console.log(Number(true));    // 1
console.log(Number(false));   // 0

Si la conversión no tiene sentido, devuelve NaN.

console.log(Number("hola")); // NaN

NaN significa Not a Number.

4.2. Convertir enteros: parseInt()

parseInt() convierte una cadena a número entero.

console.log(parseInt("42", 10));    // 42
console.log(parseInt("42px", 10));  // 42
console.log(parseInt("3.99", 10));  // 3

El segundo argumento 10 indica que queremos interpretar el número en base decimal.

Caso importante:

console.log(parseInt("px42", 10)); // NaN

parseInt() empieza a leer desde el principio. Si el texto no empieza por un número, la conversión falla.

4.3. Convertir decimales: parseFloat()

parseFloat() convierte una cadena a número decimal.

console.log(parseFloat("3.14"));   // 3.14
console.log(parseFloat("3.14€"));  // 3.14
console.log(parseFloat("10"));     // 10

Se usa cuando necesitamos conservar decimales:

const precioTexto = "19.99";
const precio = parseFloat(precioTexto);

console.log(precio + 5); // 24.99

4.4. Comprobar si una conversión ha fallado: Number.isNaN()

Podemos comprobar si un valor es NaN con Number.isNaN().

Recuerda Primero convertimos el dato con Number() y después comprobamos el resultado con Number.isNaN().

const edad = Number("hola");

console.log(Number.isNaN(edad)); // true

Ejemplo práctico:

const dato = "25";
const numero = Number(dato);

if (Number.isNaN(numero)) {
  console.log("El dato no es un número válido");
} else {
  console.log("Número introducido:", numero);
}

4.5. Formatear decimales: .toFixed()

El método .toFixed() devuelve un número formateado con una cantidad fija de decimales.

const precio = 19.987;

console.log(precio.toFixed(2)); // "19.99"

Atención: .toFixed() devuelve una cadena de texto, no un número.

Ejemplo:

const precio = 10;
const precioFormateado = precio.toFixed(2);

console.log(precioFormateado);        // "10.00"
console.log(typeof precioFormateado); // "string"

5. Math: operaciones matemáticas

Math es un objeto predefinido que agrupa funciones y constantes matemáticas.

No se crea con new. Se usa directamente:

console.log(Math.PI);

5.1. Constante Math.PI

Math.PI contiene el valor aproximado de π.

console.log(Math.PI); // 3.141592653589793

Ejemplo:

const radio = 5;
const area = Math.PI * radio * radio;

console.log(area);

5.2. Redondeos: Math.round(), Math.floor() y Math.ceil()

Math.round() redondea al entero más cercano.

console.log(Math.round(3.4)); // 3
console.log(Math.round(3.6)); // 4

Math.floor() redondea hacia abajo.

console.log(Math.floor(3.9)); // 3

Math.ceil() redondea hacia arriba.

console.log(Math.ceil(3.1)); // 4

Tabla resumen:

Método Resultado
Math.round(3.6) 4
Math.floor(3.9) 3
Math.ceil(3.1) 4

5.3. Máximo y mínimo: Math.max() y Math.min()

Math.max() devuelve el mayor de varios números.

console.log(Math.max(4, 7, 2, 9)); // 9

Math.min() devuelve el menor de varios números.

console.log(Math.min(4, 7, 2, 9)); // 2

Ejemplo práctico:

const nota1 = 6;
const nota2 = 8;
const nota3 = 5;

const notaMaxima = Math.max(nota1, nota2, nota3);

console.log("Nota más alta:", notaMaxima);

5.4. Números aleatorios: Math.random()

Math.random() devuelve un número pseudoaleatorio entre 0 incluido y 1 excluido.

const r = Math.random();

console.log(r); // Por ejemplo: 0.4728391

Importante Math.random() sirve para ejemplos, juegos sencillos o simulaciones básicas. No debe usarse para generar contraseñas ni datos de seguridad.

Para generar un entero entre 1 y 10:

const aleatorio = Math.floor(Math.random() * 10) + 1;

console.log(aleatorio);

Explicación:

Math.random() * 10

genera un número decimal entre 0 y menor que 10.

Math.floor(Math.random() * 10)

lo convierte en un entero entre 0 y 9.

Math.floor(Math.random() * 10) + 1

lo desplaza al rango 1 a 10.

5.5. Función para generar enteros aleatorios

Podemos encapsular la lógica anterior en una función:

function numeroAleatorioEntre(min, max) {
  return Math.floor(Math.random() * (max - min + 1)) + min;
}

console.log(numeroAleatorioEntre(1, 6));  // Simula un dado
console.log(numeroAleatorioEntre(1, 10)); // Número entre 1 y 10

Esta función devuelve un número entero entre min y max, ambos incluidos.

6. Date: fechas y horas

Date permite trabajar con fechas y horas.

A diferencia de Math, para crear una fecha sí usamos new.

const ahora = new Date();

console.log(ahora);

Esto crea un objeto Date con la fecha y hora actuales del sistema.

6.1. Crear la fecha actual

const hoy = new Date();

console.log(hoy);

El resultado puede mostrarse con mucha información, porque incluye fecha, hora y zona horaria.

6.2. Obtener año, mes y día

Podemos obtener partes concretas de una fecha:

const hoy = new Date();

const anyo = hoy.getFullYear();
const mes = hoy.getMonth() + 1;
const dia = hoy.getDate();

console.log("Hoy es:", dia + "/" + mes + "/" + anyo);

Métodos usados:

Método Qué devuelve
getFullYear() Año completo
getMonth() Mes de 0 a 11
getDate() Día del mes

Importante: getMonth() devuelve el mes empezando en 0. Enero es 0 y diciembre es 11. Por eso solemos sumar 1.

6.3. Obtener día de la semana

getDay() devuelve el día de la semana como número:

Valor Día
0 Domingo
1 Lunes
2 Martes
3 Miércoles
4 Jueves
5 Viernes
6 Sábado

Ejemplo:

const hoy = new Date();

console.log(hoy.getDay());

Podemos usar un array para mostrar el nombre del día:

const dias = ["Domingo", "Lunes", "Martes", "Miércoles", "Jueves", "Viernes", "Sábado"];

const hoy = new Date();
const diaSemana = hoy.getDay();

console.log("Hoy es", dias[diaSemana]);

6.4. Obtener hora y minutos

const ahora = new Date();

const hora = ahora.getHours();
const minutos = ahora.getMinutes();

console.log("Hora actual:", hora + ":" + minutos);

Podemos mejorar el formato de los minutos usando padStart():

const ahora = new Date();

const hora = ahora.getHours();
const minutos = String(ahora.getMinutes()).padStart(2, "0");

console.log("Hora actual:", hora + ":" + minutos);

padStart(2, "0") rellena por la izquierda hasta tener dos caracteres.

6.5. Precauciones con Date

Al trabajar con fechas, conviene recordar:

  • getMonth() empieza en 0.
  • getDay() empieza en domingo.
  • La fecha y hora dependen del sistema y de la zona horaria.
  • Para este nivel vamos a usar Date solo para operaciones sencillas.

7. JSON: datos en formato texto

JSON significa JavaScript Object Notation.

Es un formato de texto que se usa mucho para guardar o enviar datos. Más adelante, cuando trabajemos con APIs y comunicación asíncrona, aparecerá constantemente.

En este bloque solo veremos dos operaciones básicas:

  • convertir datos de JavaScript a texto JSON;
  • convertir texto JSON a datos de JavaScript.

7.1. Convertir datos a texto JSON: JSON.stringify()

JSON.stringify() convierte un dato de JavaScript en una cadena JSON.

Ejemplo con un array:

const notas = [4, 7, 9];

const textoJSON = JSON.stringify(notas);

console.log(textoJSON);        // "[4,7,9]"
console.log(typeof textoJSON); // "string"

Ejemplo con un objeto sencillo:

const alumno = {
  nombre: "Ana",
  nota: 8
};

const textoJSON = JSON.stringify(alumno);

console.log(textoJSON);        // {"nombre":"Ana","nota":8}
console.log(typeof textoJSON); // "string"

No estudiaremos todavía objetos propios en profundidad. En este ejemplo basta con entender que { nombre: "Ana", nota: 8 } agrupa datos relacionados.

7.2. Convertir texto JSON a datos de JavaScript: JSON.parse()

JSON.parse() convierte una cadena JSON válida en un dato de JavaScript.

const textoJSON = '{"nombre":"Ana","nota":8}';

const alumno = JSON.parse(textoJSON);

console.log(alumno.nombre); // "Ana"
console.log(alumno.nota);   // 8

Ejemplo con un array:

const textoNotas = "[4,7,9]";

const notas = JSON.parse(textoNotas);

console.log(notas[0]);      // 4
console.log(notas.length);  // 3

7.3. El texto JSON debe estar bien escrito

Para que JSON.parse() funcione, el texto debe ser JSON válido.

Correcto:

const texto = '{"nombre":"Ana","nota":8}';

Incorrecto:

const texto = "{nombre:'Ana', nota:8}";

En JSON:

  • las claves deben ir entre comillas dobles;
  • las cadenas de texto deben ir entre comillas dobles;
  • no se permiten comentarios;
  • no se pueden dejar comas finales.

7.4. Para qué servirá JSON más adelante

JSON será importante cuando trabajemos con:

  • datos recibidos desde una API;
  • comunicación asíncrona;
  • fetch;
  • aplicaciones dinámicas;
  • almacenamiento o intercambio de información.

Por ahora solo necesitamos entender estas dos ideas:

JSON.stringify(dato); // JavaScript -> texto JSON
JSON.parse(texto);    // texto JSON -> JavaScript

9. Errores frecuentes

Error 1. Pensar que un texto leído por el usuario ya es un número

const edad = "18";

console.log(edad + 1); // "181"

Corrección:

const edad = Number("18");

console.log(edad + 1); // 19

Error 2. Usar parseInt() cuando se necesitan decimales

const precio = parseInt("19.99", 10);

console.log(precio); // 19

Corrección:

const precio = parseFloat("19.99");

console.log(precio); // 19.99

Error 3. No comprobar si una conversión falla

const numero = Number("hola");

console.log(numero + 5); // NaN

Mejor:

const numero = Number("hola");

if (Number.isNaN(numero)) {
  console.log("No se puede convertir a número");
}

Error 4. Olvidar que getMonth() empieza en 0

const fecha = new Date();

console.log(fecha.getMonth()); // Enero es 0

Corrección habitual:

const mes = fecha.getMonth() + 1;

Error 5. Pensar que .toUpperCase() modifica el texto original

const texto = "dwcc";

texto.toUpperCase();

console.log(texto); // "dwcc"

Corrección:

const texto = "dwcc";

const mayusculas = texto.toUpperCase();

console.log(mayusculas); // "DWCC"

Error 6. Confundir Math.round(), Math.floor() y Math.ceil()

console.log(Math.round(3.9)); // 4
console.log(Math.floor(3.9)); // 3
console.log(Math.ceil(3.1));  // 4

Error 7. Intentar convertir texto que no es JSON válido

const texto = "{nombre:'Ana'}";

const dato = JSON.parse(texto); // Error

Corrección:

const texto = '{"nombre":"Ana"}';

const dato = JSON.parse(texto);

console.log(dato.nombre);

12. Resumen final

En este bloque hemos visto varios objetos y herramientas predefinidas de JavaScript:

  • String permite trabajar con textos mediante propiedades y métodos como .length, .toUpperCase(), .toLowerCase(), .trim(), .includes(), .startsWith(), .endsWith() y .slice().
  • Number(), parseInt() y parseFloat() permiten convertir datos a número.
  • Number.isNaN() permite comprobar si una conversión ha fallado.
  • Math permite hacer operaciones matemáticas, redondeos, obtener máximos y mínimos y generar números aleatorios.
  • Date permite obtener la fecha y la hora actual y extraer partes como año, mes, día, hora o minutos.
  • JSON permite convertir datos de JavaScript a texto JSON y recuperar datos a partir de texto JSON.

Estos conocimientos serán muy útiles cuando avancemos hacia formularios, DOM, eventos, APIs y aplicaciones dinámicas.
