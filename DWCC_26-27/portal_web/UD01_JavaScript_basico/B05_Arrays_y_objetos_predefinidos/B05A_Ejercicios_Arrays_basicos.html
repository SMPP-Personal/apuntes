Bloque 5A. Hoja de ejercicios: arrays básicos

Indicaciones generales

En esta hoja se practican los contenidos básicos de arrays en JavaScript:

  • creación de arrays;
  • acceso mediante índices;
  • modificación de elementos;
  • uso de length;
  • recorrido con for;
  • acumuladores y contadores;
  • búsqueda básica;
  • métodos básicos sin callbacks;
  • funciones que reciben arrays.

En esta hoja no se deben usar todavía forEach, map, filter, reduce ni funciones flecha como callbacks. Esos contenidos se trabajarán en el bloque B05B.

Parte 1. Crear arrays y acceder a elementos

Ejercicio 1. Crear un array de nombres

  1. Crea un array llamado nombres con al menos 4 nombres de personas.
  2. Muestra por consola:
    • el primer nombre;
    • el segundo nombre;
    • el último nombre, sin escribir el índice a mano.

Ejemplo de inicio:

const nombres = ["Ana", "Luis", "María", "Noa"];

Recuerda que el último elemento se obtiene con:

nombres[nombres.length - 1]

Ejercicio 2. Array de notas

  1. Crea un array notas con 5 números.
  2. Muestra por consola:
    • la nota de la posición 0;
    • la nota de la posición 2;
    • la nota de la posición 4.
  3. Cambia la nota de la posición 0 por un 6.
  4. Muestra el array completo antes y después del cambio.

Ejemplo de inicio:

const notas = [4, 7, 9, 3, 5];

console.log("Notas iniciales:", notas);

// Completa aquí el acceso a posiciones y la modificación.

console.log("Notas modificadas:", notas);

Ejercicio 3. Primer y último elemento

Crea un array llamado precios con al menos 4 precios.

Después muestra:

  • el primer precio;
  • el último precio;
  • un mensaje con el siguiente formato:
Primer precio: 12
Último precio: 8

No escribas a mano el índice del último elemento. Usa precios.length - 1.

Parte 2. Modificar elementos y entender const con arrays

Ejercicio 4. Modificar elementos de un array

Dado el siguiente array:

const productos = ["ratón", "teclado", "monitor"];
  1. Muestra el array completo.
  2. Cambia "teclado" por "teclado mecánico".
  3. Muestra de nuevo el array.
  4. Explica en un comentario por qué se puede modificar el contenido aunque el array esté declarado con const.

Ejercicio 5. Detectar operaciones permitidas y no permitidas

Observa el siguiente código:

const numeros = [1, 2, 3];

numeros[0] = 10;
numeros.push(4);
numeros = [7, 8, 9];

Responde:

  1. ¿Qué líneas son válidas?
  2. ¿Qué línea produce error?
  3. ¿Por qué?

Después, copia el código en la consola o en un archivo .js, comenta la línea incorrecta y comprueba el resultado.

Ejercicio 16. Comprobar si existe un valor con includes()

Dado el array:

const usuarios = ["ana", "luis", "maria", "noa"];
  1. Crea una constante usuarioBuscado.
  2. Usa includes() para comprobar si está en el array.
  3. Muestra un mensaje distinto según el resultado.

Ejemplo:

const usuarioBuscado = "luis";

if (usuarios.includes(usuarioBuscado)) {
  console.log("Usuario encontrado");
} else {
  console.log("Usuario no encontrado");
}

Parte 7. Corrige los errores

Ejercicio 21. Error con el límite del bucle

El siguiente código muestra un undefined al final. Corrígelo y explica el error en un comentario. 

const notas = [4, 7, 9];

for (let i = 0; i <= notas.length; i++) {
  console.log(notas[i]);
}

Ejercicio 22. Error por confundir índice y valor

El siguiente código pretende sumar los valores del array, pero no lo hace correctamente. Corrígelo. 

const numeros = [10, 20, 30];
let suma = 0;

for (let i = 0; i < numeros.length; i++) {
  suma += i;
}

console.log(suma);

Ejercicio 23. Error con const y arrays

El siguiente código produce un error. Corrígelo de dos formas distintas: 

const nombres = ["Ana", "Luis"];

nombres = ["María", "Noa"];

Opciones posibles:

  1. Mantener const y modificar posiciones del array.
  2. Cambiar a let si realmente se quiere reasignar todo el array.
