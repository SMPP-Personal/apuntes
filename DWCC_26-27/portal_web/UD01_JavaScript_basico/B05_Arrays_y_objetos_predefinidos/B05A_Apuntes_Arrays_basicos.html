Bloque 5A. Arrays básicos en JavaScript

1. Qué vas a aprender en este bloque

En este bloque vas a trabajar con arrays, una de las estructuras de datos más utilizadas en JavaScript.

Un array permite guardar varios valores relacionados dentro de una sola variable. Hasta ahora hemos trabajado con variables individuales: 

const nota1 = 4;
const nota2 = 7;
const nota3 = 9;

Con un array podemos agrupar esos valores:

const notas = [4, 7, 9];

Esto permite recorrer los datos, calcular resultados, buscar valores y organizar mejor el código.

En este bloque aprenderás a:

  • crear arrays;
  • acceder a sus elementos mediante índices;
  • modificar elementos;
  • usar la propiedad length;
  • recorrer arrays con for;
  • aplicar patrones básicos como sumar, contar, buscar o calcular medias;
  • utilizar métodos básicos como push(), pop(), shift(), unshift(), includes() e indexOf();
  • crear funciones que reciben arrays como parámetro.

2. Qué es un array

Un array es una estructura de datos que permite guardar varios valores en una sola variable, como si fuera una lista ordenada. 

const notas = [4, 7, 9, 3, 5];

En este ejemplo:

  • notas es el nombre del array;
  • el array contiene 5 valores;
  • cada valor ocupa una posición;
  • las posiciones se llaman índices.

Los índices en JavaScript empiezan en 0:

Elemento Valor Índice
Primer elemento 4 0
Segundo elemento 7 1
Tercer elemento 9 2
Cuarto elemento 3 3
Quinto elemento 5 4

Si un array tiene 5 elementos, su último índice no es 5, sino 4.

3. Crear arrays

3.1. Crear un array con valores

La forma recomendada para crear arrays en JavaScript es usar corchetes []. 

const numeros = [10, 20, 30];
const nombres = ["Ana", "Luis", "María"];
const aprobados = [true, false, true];

Cada elemento se separa con una coma.

3.2. Arrays con distintos tipos de datos

JavaScript permite crear arrays con valores de distinto tipo: 

const mezcla = [1, "hola", true, null];

Técnicamente es válido, pero en código real suele ser más claro que un array guarde valores del mismo tipo o relacionados entre sí.

Ejemplo recomendable:

const notas = [4, 7, 9, 3, 5];

Ejemplo menos claro:

const datos = [4, "Ana", true, null];

3.3. Crear un array vacío

También podemos crear un array vacío y añadir datos después: 

const tareas = [];

Más adelante podemos añadir elementos con métodos como push(). 

tareas.push("Repasar JavaScript");
tareas.push("Entregar práctica");

console.log(tareas);

4. Acceder a los elementos de un array

Para acceder a un elemento usamos el nombre del array y su índice entre corchetes. 

const notas = [4, 7, 9, 3, 5];

console.log(notas[0]); // 4
console.log(notas[1]); // 7
console.log(notas[4]); // 5

Recuerda:

  • notas[0] accede al primer elemento;
  • notas[1] accede al segundo;
  • notas[4] accede al quinto.

4.1. Acceder a una posición inexistente

Si intentamos acceder a una posición que no existe, JavaScript devuelve undefined. 

const notas = [4, 7, 9];

console.log(notas[0]);  // 4
console.log(notas[10]); // undefined

Esto no detiene el programa, pero normalmente indica que estamos intentando acceder a una posición incorrecta.

5. Modificar elementos de un array

Podemos cambiar el valor de una posición concreta usando su índice. 

const notas = [4, 7, 9];

console.log(notas); // [4, 7, 9]

notas[0] = 6;

console.log(notas); // [6, 7, 9]

Aunque el array esté declarado con const, podemos modificar sus elementos internos.

Lo que no podemos hacer es reasignar toda la constante:

const notas = [4, 7, 9];

notas[0] = 6;       // Correcto
// notas = [10, 8]; // Error: no se puede reasignar una constante

La constante notas sigue apuntando al mismo array. Lo que cambia es el contenido del array.

6. Longitud del array: length

La propiedad length indica cuántos elementos tiene un array. 

const notas = [4, 7, 9, 3, 5];

console.log(notas.length); // 5

length es muy útil para recorrer arrays sin escribir a mano cuántos elementos hay.

No es recomendable hacer esto:

const notas = [4, 7, 9, 3, 5];

for (let i = 0; i < 5; i++) {
  console.log(notas[i]);
}

Es mejor usar length:

const notas = [4, 7, 9, 3, 5];

for (let i = 0; i < notas.length; i++) {
  console.log(notas[i]);
}

Así el código seguirá funcionando aunque el array tenga más o menos elementos.

7. Primer y último elemento

El primer elemento de un array está en la posición 0. 

const nombres = ["Ana", "Luis", "María"];

console.log(nombres[0]); // "Ana"

El último elemento está en la posición length - 1. 

const nombres = ["Ana", "Luis", "María"];

console.log(nombres[nombres.length - 1]); // "María"

Esto funciona aunque el array cambie de tamaño.

const frutas = ["manzana", "pera", "naranja", "kiwi"];

console.log(frutas[frutas.length - 1]); // "kiwi"

8. Recorrer un array con for

Recorrer un array significa visitar sus elementos uno a uno.

La forma clásica es usar un bucle for con un índice. 

const notas = [4, 7, 9, 3, 5];

for (let i = 0; i < notas.length; i++) {
  console.log("Índice:", i, "Valor:", notas[i]);
}

En este ejemplo:

  • i empieza en 0;
  • el bucle se repite mientras i < notas.length;
  • en cada vuelta accedemos al elemento con notas[i].

8.1. Error frecuente: usar <=

Este código tiene un error:

const notas = [4, 7, 9];

for (let i = 0; i <= notas.length; i++) {
  console.log(notas[i]);
}

El problema es que i <= notas.length permite llegar hasta i = 3, pero los índices válidos son 0, 1 y 2.

La versión correcta es:

const notas = [4, 7, 9];

for (let i = 0; i < notas.length; i++) {
  console.log(notas[i]);
}

9. Patrones básicos con arrays

Una vez sabemos recorrer arrays, podemos hacer operaciones útiles con sus elementos.

9.1. Sumar los valores de un array

const numeros = [4, 7, 9, 3, 5];
let suma = 0;

for (let i = 0; i < numeros.length; i++) {
  suma += numeros[i];
}

console.log("Suma:", suma);

Funcionamiento:

  • suma empieza en 0;
  • en cada vuelta se añade un número del array;
  • al final, suma contiene el total.

9.2. Calcular la media

Para calcular la media necesitamos sumar los valores y dividir entre el número de elementos. 

const notas = [4, 7, 9, 3, 5];
let suma = 0;

for (let i = 0; i < notas.length; i++) {
  suma += notas[i];
}

const media = suma / notas.length;

console.log("Media:", media);

Antes de calcular una media conviene asegurarse de que el array no está vacío. 

const notas = [];

if (notas.length > 0) {
  let suma = 0;

  for (let i = 0; i < notas.length; i++) {
    suma += notas[i];
  }

  const media = suma / notas.length;
  console.log("Media:", media);
} else {
  console.log("No hay notas para calcular la media");
}

9.3. Contar elementos que cumplen una condición

Podemos contar cuántos elementos cumplen una condición. Por ejemplo, cuántas notas son aprobadas. 

const notas = [4, 7, 9, 3, 5];
let aprobados = 0;

for (let i = 0; i < notas.length; i++) {
  if (notas[i] >= 5) {
    aprobados++;
  }
}

console.log("Aprobados:", aprobados);

También podemos contar suspensos:

const notas = [4, 7, 9, 3, 5];
let suspensos = 0;

for (let i = 0; i < notas.length; i++) {
  if (notas[i] < 5) {
    suspensos++;
  }
}

console.log("Suspensos:", suspensos);

9.4. Buscar un valor manualmente

Podemos recorrer un array para comprobar si contiene un valor. 

const nombres = ["Ana", "Luis", "María"];
let encontrado = false;

for (let i = 0; i < nombres.length; i++) {
  if (nombres[i] === "Luis") {
    encontrado = true;
  }
}

if (encontrado) {
  console.log("Luis está en la lista");
} else {
  console.log("Luis no está en la lista");
}

Más adelante veremos métodos que permiten hacer esto de forma más directa.

10. Métodos básicos de arrays

Los arrays tienen métodos incorporados que permiten realizar operaciones frecuentes.

En este bloque veremos métodos sencillos. Los métodos avanzados con callbacks se estudiarán en el siguiente bloque.

10.1. push(): añadir al final

push() añade un elemento al final del array.

const frutas = ["manzana", "pera"];

frutas.push("naranja");

console.log(frutas); // ["manzana", "pera", "naranja"]

push() modifica el array original.

10.2. pop(): eliminar el último elemento

pop() elimina el último elemento del array y lo devuelve. 

const frutas = ["manzana", "pera", "naranja"];

const ultima = frutas.pop();

console.log(ultima); // "naranja"
console.log(frutas); // ["manzana", "pera"]

10.3. unshift(): añadir al principio

unshift() añade un elemento al inicio del array. 

const frutas = ["pera", "naranja"];

frutas.unshift("manzana");

console.log(frutas); // ["manzana", "pera", "naranja"]

10.4. shift(): eliminar el primer elemento

shift() elimina el primer elemento del array y lo devuelve. 

const frutas = ["manzana", "pera", "naranja"];

const primera = frutas.shift();

console.log(primera); // "manzana"
console.log(frutas);  // ["pera", "naranja"]

10.5. includes(): comprobar si existe un valor

includes() devuelve true si el array contiene el valor indicado. 

const nombres = ["Ana", "Luis", "María"];

console.log(nombres.includes("Luis"));  // true
console.log(nombres.includes("Pedro")); // false

Este método es útil cuando solo necesitamos saber si un valor existe o no.

10.6. indexOf(): obtener la posición de un valor

indexOf() devuelve el índice donde aparece un valor. Si no lo encuentra, devuelve -1. 

const nombres = ["Ana", "Luis", "María"];

console.log(nombres.indexOf("Luis"));  // 1
console.log(nombres.indexOf("Pedro")); // -1

Podemos usarlo en una condición:

const nombres = ["Ana", "Luis", "María"];

if (nombres.indexOf("Pedro") === -1) {
  console.log("Pedro no está en la lista");
}

11. Arrays y funciones

Como ya hemos trabajado funciones, podemos crear funciones que reciban arrays como argumento.

Esto permite reutilizar código con distintas listas de datos.

11.1. Función que muestra todos los elementos

function mostrarElementos(lista) {
  for (let i = 0; i < lista.length; i++) {
    console.log(lista[i]);
  }
}

const nombres = ["Ana", "Luis", "María"];

mostrarElementos(nombres);

11.2. Función que suma los valores de un array

function sumarArray(numeros) {
  let suma = 0;

  for (let i = 0; i < numeros.length; i++) {
    suma += numeros[i];
  }

  return suma;
}

const notas = [4, 7, 9, 3, 5];

console.log(sumarArray(notas)); // 28

11.3. Función que calcula la media

function calcularMedia(numeros) {
  if (numeros.length === 0) {
    return null;
  }

  let suma = 0;

  for (let i = 0; i < numeros.length; i++) {
    suma += numeros[i];
  }

  return suma / numeros.length;
}

const notas = [4, 7, 9, 3, 5];
const media = calcularMedia(notas);

if (media !== null) {
  console.log("Media:", media);
} else {
  console.log("No se puede calcular la media de un array vacío");
}

11.4. Función que cuenta aprobados

function contarAprobados(notas) {
  let aprobados = 0;

  for (let i = 0; i < notas.length; i++) {
    if (notas[i] >= 5) {
      aprobados++;
    }
  }

  return aprobados;
}

const notasClase = [4, 7, 9, 3, 5];

console.log("Aprobados:", contarAprobados(notasClase));

12. Tabla resumen de métodos básicos

Método Qué hace Modifica el array original

pop() Elimina el último elemento Sí

shift() Elimina el primer elemento Sí
includes(valor) Comprueba si un valor existe No

13. Errores frecuentes

13.1. Pensar que el primer índice es 1

Incorrecto:

const colores = ["rojo", "verde", "azul"];

console.log(colores[1]); // "verde", no "rojo"

Correcto:

const colores = ["rojo", "verde", "azul"];

console.log(colores[0]); // "rojo"

13.2. Usar <= al recorrer el array

Incorrecto:

const datos = [10, 20, 30];

for (let i = 0; i <= datos.length; i++) {
  console.log(datos[i]);
}

El último valor mostrado será undefined.

Correcto:

const datos = [10, 20, 30];

for (let i = 0; i < datos.length; i++) {
  console.log(datos[i]);
}

13.3. Confundir const con inmutabilidad

const frutas = ["manzana", "pera"];

frutas.push("naranja"); // Correcto

// frutas = ["kiwi"];   // Error

const impide reasignar la variable, pero no impide modificar el contenido del array.

13.4. Olvidar inicializar acumuladores o contadores

Incorrecto:

const numeros = [1, 2, 3];
let suma;

for (let i = 0; i < numeros.length; i++) {
  suma += numeros[i];
}

console.log(suma); // NaN

Correcto:

const numeros = [1, 2, 3];
let suma = 0;

for (let i = 0; i < numeros.length; i++) {
  suma += numeros[i];
}

console.log(suma); // 6

13.5. Acceder a posiciones que no existen

const nombres = ["Ana", "Luis"];

console.log(nombres[5]); // undefined

Esto suele indicar que el índice utilizado no es correcto.

16. Resumen final

Un array permite guardar varios valores en una sola variable.

Ideas clave:

  • Los arrays se crean con corchetes [].
  • Los elementos se separan con comas.
  • Los índices empiezan en 0.
  • array[0] accede al primer elemento.
  • array.length indica cuántos elementos tiene.
  • El último elemento está en array[array.length - 1].
  • Podemos recorrer un array con un for.
  • Los arrays declarados con const pueden modificar su contenido, pero no pueden reasignarse.
  • Los métodos básicos push(), pop(), unshift(), shift(), includes() e indexOf() permiten manipular o consultar arrays sencillos.
  • Las funciones pueden recibir arrays como parámetros y devolver resultados calculados a partir de sus elementos.
