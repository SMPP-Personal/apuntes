Bloque 5C. Ejercicios: objetos predefinidos en JavaScript

1. Objetivo de la hoja

En esta hoja vas a practicar algunos objetos y herramientas predefinidas de JavaScript que se utilizan con mucha frecuencia en desarrollo web:

  • String, para trabajar con textos.
  • Conversión numérica y Number, para pasar datos de texto a número y validar conversiones.
  • Math, para cálculos matemáticos, redondeos y números aleatorios.
  • Date, para obtener fechas y horas.
  • JSON, para convertir datos de JavaScript a texto y recuperar datos desde texto.

En esta hoja no se trabajan todavía DOM, eventos, formularios reales, fetch, promesas ni async/await. Eso se verá en documentos posteriores.

2. Instrucciones generales

  1. Crea un archivo llamado B05C_ejercicios_objetos_predefinidos.js.
  2. Resuelve los ejercicios en orden.
  3. Usa console.log() para comprobar los resultados.
  4. Lee bien si el ejercicio pide predecir, probar, explicar o crear código.
  5. Añade comentarios breves cuando el ejercicio pida justificar una respuesta.

Ejemplo de comentario:

// getMonth() devuelve valores entre 0 y 11, por eso sumamos 1.

Parte 1. String: trabajar con texto

Ejercicio 1. Longitud, mayúsculas y minúsculas

Dada la cadena:

const mensaje = "Desenvolvemento Web en Contorno Cliente";
  1. Muestra por consola la longitud del mensaje usando .length.
  2. Muestra el mensaje completo en mayúsculas usando .toUpperCase().
  3. Muestra el mensaje completo en minúsculas usando .toLowerCase().
  4. Comprueba si el valor original de mensaje ha cambiado.

Ejercicio 2. Limpiar espacios con trim()

Dado este valor:

const nombreConEspacios = "   Ana   ";
  1. Muestra por consola el valor original.
  2. Crea una constante nombreLimpio aplicando .trim().
  3. Muestra nombreLimpio por consola.
  4. Muestra la longitud del texto original y la longitud del texto limpio.
  5. Explica en un comentario qué elimina exactamente trim().

Ejercicio 3. Buscar texto con includes()

Dado:

const descripcion = "Curso de DWCC con JavaScript y HTML";
  1. Comprueba si la descripción contiene la palabra "DWCC" usando .includes().
  2. Comprueba si contiene la palabra "PHP".
  3. Escribe un if que muestre "El curso es de DWCC" solo si .includes("DWCC") devuelve true.
  4. Prueba ahora con descripcion.includes("dwcc"). ¿Da el mismo resultado? Explica por qué.

Ejercicio 4. Búsqueda sin distinguir mayúsculas y minúsculas

Dado:

const modulo = "Desenvolvemento Web en Contorno Cliente";
const busqueda = "web";
  1. Convierte modulo a minúsculas.
  2. Convierte busqueda a minúsculas.
  3. Comprueba si modulo contiene el texto de busqueda, sin importar las mayúsculas o minúsculas.
  4. Muestra un mensaje indicando si se ha encontrado la búsqueda.

Ejercicio 5. Comprobar inicio y final de una cadena

Dado:

const archivo = "practica_arrays.js";
  1. Comprueba si el nombre del archivo empieza por "practica" usando .startsWith().
  2. Comprueba si termina en ".js" usando .endsWith().
  3. Escribe un if que muestre "Archivo JavaScript correcto" solo si se cumplen las dos condiciones.

Ejercicio 6. Extraer parte de un texto con slice()

Dado:

const codigo = "DWCC-2026-A";
  1. Extrae los cuatro primeros caracteres.
  2. Extrae el año "2026".
  3. Extrae la última letra.
  4. Muestra todos los resultados por consola.

Pista: para extraer la última letra puedes usar codigo.slice(codigo.length - 1).

Pista: recuerda que los índices empiezan en 0.

Parte 2. Conversión numérica y Number

Ejercicio 7. Convertir cadenas a números con Number()

Antes de ejecutar, intenta predecir el resultado de cada línea. Después compruébalo en la consola.

console.log(Number("123"));
console.log(Number("3.14"));
console.log(Number("hola"));
console.log(Number(true));
console.log(Number(false));
console.log(Number(""));
  1. Escribe el resultado esperado de cada línea en comentarios.
  2. Explica qué significa NaN.
  3. Comprueba con Number.isNaN() cuáles de esos resultados son NaN.

Ejercicio 8. parseInt() y parseFloat()

Predice primero y comprueba después:

console.log(parseInt("42", 10));
console.log(parseInt("42px", 10));
console.log(parseInt("px42", 10));

console.log(parseFloat("3.14"));
console.log(parseFloat("3.14€"));
console.log(parseFloat("hola 2.5"));
  1. Indica el resultado de cada línea.
  2. Explica la diferencia entre parseInt() y parseFloat().
  3. Explica por qué parseInt("42px", 10) sí devuelve un número y parseInt("px42", 10) no.

Ejercicio 9. Calcular precio con IVA

Supón que recibes el precio de un producto como texto:

const precioTexto = "19.99";
  1. Convierte precioTexto a número decimal usando parseFloat().
  2. Calcula el precio con IVA del 21%.
  3. Muestra el resultado por consola.
  4. Muestra el resultado con dos decimales usando .toFixed(2).
  5. Prueba qué ocurre si precioTexto vale "19.99€".

Ejercicio 10. Validar una edad introducida como texto

Dado:

const edadTexto = "18";
  1. Convierte edadTexto a número con Number().
  2. Comprueba si la conversión ha fallado usando Number.isNaN().
  3. Si la conversión es válida y la edad es mayor o igual que 18, muestra "Mayor de edad".
  4. Si la conversión es válida y la edad es menor que 18, muestra "Menor de edad".
  5. Si la conversión no es válida, muestra "Edad no válida".
  6. Prueba el ejercicio cambiando edadTexto por "hola".

Ejercicio 11. Redondear un importe con toFixed()

Dado:

const total = 12.5 / 3;
  1. Muestra el valor original de total.
  2. Muestra el valor con dos decimales usando .toFixed(2).
  3. Comprueba con typeof qué tipo de dato devuelve .toFixed(2).
  4. Explica el resultado en un comentario.

Parte 3. Math: cálculos matemáticos

Ejercicio 12. Redondeos con Math

console.log(Math.round(3.4));
console.log(Math.round(3.5));
console.log(Math.round(3.6));

console.log(Math.floor(3.9));
console.log(Math.ceil(3.1));
  1. Indica el resultado de cada línea.
  2. Explica con tus palabras la diferencia entre round, floor y ceil.

Ejercicio 13. Nota redondeada

Dada esta nota:

const nota = 6.75;
  1. Usa Math.round() para redondear a la unidad más cercana.
  2. Usa Math.floor() para redondear hacia abajo.
  3. Usa Math.ceil() para redondear hacia arriba.
  4. Muestra cada resultado con un mensaje explicativo.

Ejercicio 14. Máximo y mínimo

Dadas estas notas:

const nota2 = 7;
const nota3 = 9;
  1. Usa Math.max() para obtener la nota más alta.
  2. Usa Math.min() para obtener la nota más baja.
  3. Muestra ambos resultados con mensajes claros.

Ejercicio 15. Tirada de dado

Escribe código que:

  1. Genere un número entero aleatorio entre 1 y 6, ambos incluidos.
  2. Muestre por consola "Has sacado un X", donde X es el número generado.

Pista:

const dado = Math.floor(Math.random() * 6) + 1;

Ejercicio 16. Número aleatorio entre dos valores

Crea una función llamada numeroAleatorioEntre(min, max) que devuelva un número entero aleatorio entre min y max, ambos incluidos.

Ejemplo de uso:

console.log(numeroAleatorioEntre(1, 10));
console.log(numeroAleatorioEntre(20, 30));

Después responde en un comentario:

  1. ¿Por qué se usa Math.floor()?
  2. ¿Para qué sirve max - min + 1?

Ejercicio 17. Área de un círculo

Crea una función llamada calcularAreaCirculo(radio) que:

  1. Reciba el radio de un círculo.
  2. Calcule el área usando Math.PI.
  3. Devuelva el resultado.
  4. Muestra el resultado con dos decimales.

Fórmula:

area = Math.PI * radio * radio;

Parte 4. Date: fechas y horas

Ejercicio 18. Mostrar la fecha actual

  1. Crea una constante hoy con la fecha actual:
const hoy = new Date();
  1. Obtén:
    • el año con .getFullYear();
    • el mes con .getMonth();
    • el día con .getDate().
  2. Corrige el mes sumando 1.
  3. Muestra un mensaje del tipo:
Hoy es 23/5/2026
  1. Explica en un comentario por qué hay que sumar 1 al mes.

Ejercicio 19. Día de la semana

Crea un programa que muestre el número del día de la semana actual usando .getDay().

Después, usando este array:

const diasSemana = ["domingo", "lunes", "martes", "miércoles", "jueves", "viernes", "sábado"];

muestra un mensaje como:

Hoy es martes

Recuerda: .getDay() devuelve 0 para domingo, 1 para lunes, etc.

Ejercicio 20. Hora actual

  1. Crea un objeto Date con la fecha y hora actuales.
  2. Obtén la hora con .getHours().
  3. Obtén los minutos con .getMinutes().
  4. Muestra un mensaje como:
Son las 12:35

No hace falta que formatees los minutos con cero inicial si son menores de 10.

Ejercicio 21. Mensaje de bienvenida con fecha y hora

Escribe un programa que muestre un mensaje parecido a este:

Bienvenido/a. Hoy es 23/5/2026 y son las 12:35.

Debes usar:

  • new Date();
  • .getFullYear();
  • .getMonth() + 1;
  • .getDate();
  • .getHours();
  • .getMinutes().

Ejercicio 22. Edad aproximada

prompt() está disponible en el navegador. Si ejecutas este archivo con Node.js, este ejemplo no funcionará directamente sin adaptar la entrada de datos.

  1. Pide al usuario con prompt() el año de nacimiento.

Ejercicio 25. Convertir un dato estructurado a JSON

  1. Convierte alumno a texto JSON usando JSON.stringify().

No es necesario estudiar todavía en profundidad los objetos propios. Aquí se usan solo como ejemplo sencillo de dato estructurado.

Ejercicio 26. Recuperar un dato estructurado desde JSON

Dado este texto JSON:

const alumnoJSON = '{"nombre":"Luis","modulo":"DWCC","nota":7}';

  2. Muestra por consola el nombre del alumno.
  3. Muestra por consola la nota.

Ejercicio 27. JSON válido e inválido

Observa estos dos textos:

const jsonValido = '{"nombre":"Ana","nota":8}';
const jsonInvalido = "{'nombre':'Ana','nota':8}";
  1. Intenta explicar cuál de los dos tiene formato JSON correcto.

  3. No hace falta que captures el error del JSON inválido, pero sí debes explicar por qué no es correcto.

Pista: en JSON, las claves y los textos deben ir con comillas dobles.

Parte 7. Corrige los errores

Ejercicio 30. Error con String

El siguiente código pretende mostrar el texto en mayúsculas, pero después espera que la variable original haya cambiado:

const nombre = "ana";
nombre.toUpperCase();
console.log(nombre); // se esperaba "ANA"
  1. Explica por qué no funciona como se esperaba.
  2. Corrige el código.

const total = Number(precio) * 2;

const mes = hoy.getMonth();

Ejercicio 33. Error conceptual: JSON no es un objeto directamente

  1. Explica por qué no muestra "Ana".
  2. Convierte primero el texto JSON a dato de JavaScript.
  3. Muestra correctamente el nombre.

Resumen de repaso

Antes de dar por terminada la hoja, asegúrate de poder responder:

  1. ¿Qué diferencia hay entre una propiedad y un método?
  2. ¿Qué métodos de String sirven para limpiar, transformar y buscar texto?
  3. ¿Qué diferencia hay entre Number(), parseInt() y parseFloat()?
  4. ¿Para qué sirve Number.isNaN()?
  5. ¿Qué diferencia hay entre Math.round(), Math.floor() y Math.ceil()?
  6. ¿Cómo se genera un entero aleatorio entre dos valores?
  7. ¿Por qué se suma 1 a getMonth()?
  8. ¿Qué diferencia hay entre JSON.stringify() y JSON.parse()?
  9. ¿Por qué un texto JSON no se puede usar como objeto hasta convertirlo con JSON.parse()?
