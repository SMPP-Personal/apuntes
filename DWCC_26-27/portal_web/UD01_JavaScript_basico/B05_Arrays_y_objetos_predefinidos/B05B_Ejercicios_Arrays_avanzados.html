Bloque 5B. Ejercicios: métodos avanzados de arrays

1. Objetivo de esta hoja

En esta hoja vas a practicar los métodos avanzados de arrays más habituales en JavaScript:

  • forEach() para recorrer un array y ejecutar una acción con cada elemento;
  • map() para transformar todos los elementos y obtener un nuevo array;
  • filter() para seleccionar algunos elementos y obtener un nuevo array;
  • reduce() para acumular todos los elementos en un único resultado;
  • slice() para copiar o extraer partes de un array sin modificar el original.

Estos ejercicios continúan el trabajo realizado en el bloque de arrays básicos. Aquí ya no nos centraremos en índices, push(), pop(), shift() o unshift(), salvo que aparezcan puntualmente como repaso.

Idea clave: en este bloque los arrays se trabajan usando funciones como argumento. Es decir, se usan callbacks.

2. Antes de empezar

Crea un archivo llamado:

B05B_ejercicios_arrays_avanzados.js

Realiza todos los ejercicios en ese archivo, separando cada ejercicio con comentarios:

// Ejercicio 1

// Ejercicio 2

// Ejercicio 3

Puedes ejecutar el código en la consola del navegador o con Node.js, según indique la profesora.

En esta hoja puedes usar funciones flecha como callbacks:

const numeros = [1, 2, 3];

numeros.forEach((numero) => {
  console.log(numero);
});

Recuerda que una función flecha también puede escribirse en una sola línea cuando la operación es sencilla:

const dobles = numeros.map((numero) => numero * 2);

Ejercicio 6. Convertir notas numéricas en textos

Dado:

const notas = [4, 7, 9, 3, 5];

Usa map() para crear un nuevo array llamado resultados, con los textos:

  • "Aprobado" si la nota es mayor o igual que 5;
  • "Suspenso" si la nota es menor que 5.

Resultado esperado:

["Suspenso", "Aprobado", "Aprobado", "Suspenso", "Aprobado"]

Puedes hacerlo con una función flecha con llaves:

const resultados = notas.map((nota) => {
  if (nota >= 5) {
    return "Aprobado";
  } else {
    return "Suspenso";
  }
});
