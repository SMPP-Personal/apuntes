Bloque 4. Hoja de ejercicios: funciones básicas en JavaScript

1. Objetivo de la hoja

Esta hoja sirve para practicar las funciones básicas en JavaScript usando la forma clásica: 

function nombreFuncion(parametros) {
  // instrucciones
}

En estos ejercicios se trabajará:

  • declaración de funciones;
  • llamada a funciones;
  • parámetros y argumentos;
  • return;
  • diferencia entre console.log() y return;
  • variables locales y ámbito de función;
  • funciones con condicionales;
  • funciones con bucles;
  • errores frecuentes.

En esta hoja no se trabajan todavía funciones flecha, funciones anónimas ni callbacks. Esos contenidos corresponden al bloque B04b.

2. Instrucciones generales

  1. Crea un archivo llamado b04_funciones_basicas.js.
  2. Resuelve los ejercicios en orden.
  3. Prueba cada ejercicio en la consola del navegador o enlazando el archivo desde un HTML básico.
  4. Añade comentarios breves en el código cuando sea necesario.
  5. No uses funciones flecha (=>) en esta hoja.
  6. No uses arrays todavía, salvo que el profesorado lo indique expresamente.

Parte 1. Declarar y llamar funciones

Ejercicio 1. Primera función

Declara una función llamada saludar que muestre por consola el texto: 

Hola desde DWCC

Después, llama a la función dos veces.

Estructura esperada:

function saludar() {
  // código
}

saludar();
saludar();

Ejercicio 2. Función con mensaje personalizado

Declara una función llamada mostrarBienvenida que muestre por consola: 

Bienvenido/a al bloque de funciones

Después, llama a la función.

Responde en un comentario dentro del código:

// ¿La función se ejecuta cuando se declara o cuando se llama?

Ejercicio 3. Detectar el error

Observa este código:

function despedir() {
  console.log("Hasta luego");
}

// Falta algo aquí

La función está declarada, pero no se muestra nada por consola.

  1. Explica por qué.
  2. Añade la línea necesaria para ejecutar la función.

Parte 2. Parámetros y argumentos

Ejercicio 4. Saludo con parámetro

Declara una función llamada saludarPersona que reciba un parámetro nombre y muestre: 

Hola, Ana

si se llama así:

saludarPersona("Ana");

Prueba también con otros nombres.

Ejercicio 5. Mensaje con dos parámetros

Declara una función llamada mostrarAlumnoModulo que reciba dos parámetros:

  • nombre;
  • modulo.

Debe mostrar una frase como:

Ana está estudiando DWCC

Ejemplo de llamada:

mostrarAlumnoModulo("Ana", "DWCC");

Ejercicio 6. Parámetros y argumentos

Dado este código:

function multiplicar(a, b) {

Ejercicio 7. Argumento omitido

Parte 3. Funciones con return

Ejercicio 8. Sumar y devolver

Declara una función llamada sumar que reciba dos números y devuelva la suma.

Después, guarda el resultado en una constante y muéstralo por consola.

Ejemplo de uso:

const resultado = sumar(3, 4);
console.log(resultado); // 7

Ejercicio 9. Calcular el doble

Declara una función llamada calcularDoble que reciba un número y devuelva su doble.

Pruebas esperadas:

console.log(calcularDoble(4));  // 8
console.log(calcularDoble(10)); // 20

Ejercicio 10. Precio con IVA

Declara una función llamada precioConIVA que reciba:

  • precioSinIVA;
  • tipoIVA.

Debe devolver el precio final con IVA incluido.

Ejemplo:

const total = precioConIVA(100, 21);
console.log(total); // 121

Prueba también con:

precioConIVA(50, 10);
precioConIVA(20, 4);

Parte 4. console.log() frente a return

Ejercicio 11. Comparar dos funciones

Copia este código y ejecútalo:

function sumarYMostrar(a, b) {
  console.log(a + b);
}

function sumarYDevolver(a, b) {
  return a + b;
}

const resultado1 = sumarYMostrar(3, 4);
const resultado2 = sumarYDevolver(3, 4);

console.log("resultado1:", resultado1);
console.log("resultado2:", resultado2);

Responde:

  1. ¿Qué muestra sumarYMostrar(3, 4)?
  2. ¿Qué valor queda guardado en resultado1?
  3. ¿Qué valor queda guardado en resultado2?
  4. ¿Cuál es la diferencia entre mostrar un valor y devolver un valor?

Ejercicio 12. Corregir una función que no devuelve nada

Este código tiene un problema:

function calcularCuadrado(n) {
  const cuadrado = n * n;
  console.log(cuadrado);
}

const resultado = calcularCuadrado(5);
console.log("Resultado:", resultado);

La función muestra el cuadrado, pero resultado vale undefined.

Corrige la función para que devuelva el cuadrado usando return.

Ejercicio 13. Código después de return

Observa este código:

function ejemploReturn() {
  return "Primer mensaje";
  console.log("Segundo mensaje");
}

console.log(ejemploReturn());

Responde:

  1. ¿Se ejecuta el console.log("Segundo mensaje")?
  2. ¿Por qué?
  3. ¿Qué regla importante sobre return se comprueba aquí?

Parte 5. Variables locales y ámbito

Ejercicio 14. Variable local

Copia y ejecuta este código:

function crearMensaje() {
  const mensaje = "Hola";
  console.log(mensaje);
}

crearMensaje();
console.log(mensaje);

Responde:

  1. ¿Qué línea produce error?
  2. ¿Por qué mensaje no se puede usar fuera de la función?
  3. ¿Cómo se llama este concepto?

Ejercicio 15. Evitar variables globales innecesarias

Este código funciona, pero no es una buena solución:

let resultado;

function calcularTriple(n) {
  resultado = n * 3;
}

calcularTriple(5);
console.log(resultado);

Reescríbelo para que la función calcularTriple devuelva el resultado con return, sin usar una variable global llamada resultado.

Ejercicio 16. Variable local con el mismo nombre

Analiza este código:

const numero = 10;

function mostrarNumero() {
  const numero = 5;
  console.log("Dentro:", numero);
}

mostrarNumero();
console.log("Fuera:", numero);

Responde:

  1. ¿Qué muestra la consola?
  2. ¿Por qué hay dos valores distintos para numero?
  3. ¿Es recomendable repetir nombres de variables aunque estén en ámbitos distintos? Justifica la respuesta.

Parte 6. Funciones con condicionales

Ejercicio 17. Mayor de edad

Declara una función llamada esMayorDeEdad que reciba una edad y devuelva:

  • true si la edad es mayor o igual que 18;
  • false en caso contrario.

Pruebas:

console.log(esMayorDeEdad(17)); // false
console.log(esMayorDeEdad(18)); // true
console.log(esMayorDeEdad(25)); // true

Ejercicio 18. Clasificar nota

Declara una función llamada clasificarNota que reciba una nota numérica y devuelva:

  • "Sobresaliente" si la nota es mayor o igual que 9;
  • "Notable" si la nota es mayor o igual que 7;
  • "Aprobado" si la nota es mayor o igual que 5;
  • "Suspenso" si la nota es menor que 5.

Pruebas:

console.log(clasificarNota(10)); // Sobresaliente
console.log(clasificarNota(8));  // Notable
console.log(clasificarNota(6));  // Aprobado
console.log(clasificarNota(3));  // Suspenso

Ejercicio 19. Validar rango

Declara una función llamada estaEnRango que reciba tres parámetros:

  • numero;
  • minimo;
  • maximo.

Debe devolver true si numero está entre minimo y maximo, ambos incluidos. En caso contrario debe devolver false.

Ejemplo:

console.log(estaEnRango(5, 1, 10));  // true
console.log(estaEnRango(15, 1, 10)); // false

Ejercicio 20. Devolver un mensaje, no mostrarlo directamente

Declara una función llamada mensajeAcceso que reciba dos parámetros:

  • edad;
  • tienePermiso.

Debe devolver:

  • "Acceso permitido" si la persona tiene 18 años o más o si tiene permiso;
  • "Acceso denegado" en caso contrario.

Después, muestra el resultado por consola desde fuera de la función.

Ejemplo:

const mensaje = mensajeAcceso(16, true);
console.log(mensaje); // Acceso permitido

Parte 7. Funciones con bucles

Ejercicio 21. Imprimir números hasta N

Declara una función llamada imprimirHasta que reciba un número n y muestre por consola los números del 1 al n.

Ejemplo:

imprimirHasta(5);

Salida esperada:

1
2
3
4
5

Ejercicio 22. Sumar de 1 a N

Declara una función llamada sumarHasta que reciba un número n y devuelva la suma desde 1 hasta n.

Ejemplo:

console.log(sumarHasta(5)); // 15, porque 1 + 2 + 3 + 4 + 5 = 15

La función debe usar un bucle.

Ejercicio 23. Tabla de multiplicar

Declara una función llamada mostrarTabla que reciba un número n y muestre su tabla de multiplicar del 1 al 10.

Ejemplo:

mostrarTabla(5);

Salida esperada:

5 x 1 = 5
5 x 2 = 10
...
5 x 10 = 50

Ejercicio 24. Contar pares hasta N

Declara una función llamada contarParesHasta que reciba un número n y devuelva cuántos números pares hay entre 1 y n, ambos incluidos si corresponde.

Ejemplos:

console.log(contarParesHasta(10)); // 5, porque son 2, 4, 6, 8, 10
console.log(contarParesHasta(7));  // 3, porque son 2, 4, 6

Parte 8. Errores frecuentes

Ejercicio 25. Falta llamar a la función

Corrige este código para que muestre el mensaje por consola: 

function mostrarCurso() {
  console.log("DWCC");
}

Ejercicio 26. return mal situado

Este código no funciona como se espera:

function sumarHastaCinco() {
  let suma = 0;

  for (let i = 1; i <= 5; i++) {
    return suma + i;
  }
}

console.log(sumarHastaCinco());

Responde:

  1. ¿Por qué el bucle no llega hasta 5?
  2. Corrige la función para que devuelva la suma total de 1 a 5.

Ejercicio 27. Confundir parámetro con variable externa

Analiza este código:

const precio = 100;

function aplicarDescuento() {
  return precio * 0.9;
}

console.log(aplicarDescuento());

Funciona, pero depende de una variable externa.

Reescribe la función para que reciba precio como parámetro: 

function aplicarDescuento(precio) {
  // código
}

Después prueba:

console.log(aplicarDescuento(100));
console.log(aplicarDescuento(50));

Ejercicio 28. Nombre poco claro

Este código funciona, pero el nombre de la función es poco descriptivo: 

function hacer(a, b) {
  return a * b;
}
  1. Propón un nombre más adecuado para la función.
  2. Renombra también los parámetros para que el código se entienda mejor.
  3. Explica por qué tu versión es más legible.

Parte 9. Mini-entrega final

Crea un archivo llamado b04_mini_entrega_funciones.js con las siguientes funciones:

  1. calcularIVA(precio, tipoIVA)
    Devuelve el importe del IVA, no el precio final.

  2. calcularPrecioFinal(precio, tipoIVA)
    Devuelve el precio con IVA incluido.

  3. esAprobado(nota)
    Devuelve true si la nota es mayor o igual que 5 y false en caso contrario.

  4. obtenerMensajeNota(nota)
    Devuelve "Aprobado" o "Suspenso" usando la función esAprobado.

  5. mostrarCuentaAtras(n)
    Muestra por consola los números desde n hasta 1.

Después, añade pruebas al final del archivo:

console.log(calcularIVA(100, 21));
console.log(calcularPrecioFinal(100, 21));
console.log(esAprobado(7));
console.log(obtenerMensajeNota(4));
mostrarCuentaAtras(5);

Entrega:

  • archivo .js;
  • captura de consola con las pruebas ejecutadas;
  • breve comentario indicando qué función te resultó más difícil y por qué.
