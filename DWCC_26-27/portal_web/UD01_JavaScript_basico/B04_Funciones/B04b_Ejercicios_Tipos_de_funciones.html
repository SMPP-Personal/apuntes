Bloque 4b. Ejercicios: tipos de funciones en JavaScript

1. Objetivo de esta hoja

Esta hoja sirve para practicar las formas de escribir y usar funciones vistas en el bloque B04b: Tipos de funciones en JavaScript.

En estos ejercicios se trabajan:

  • funciones anónimas asignadas a constantes;
  • funciones flecha con llaves y return explícito;
  • funciones flecha compactas con retorno implícito;
  • callbacks básicos;
  • diferencia entre pasar una función y ejecutarla;
  • uso básico de setTimeout.

En esta hoja no trabajamos todavía métodos de arrays como map, filter, forEach o reduce, ni eventos del DOM, ni promesas, ni async/await.

2. Recordatorio mínimo

En JavaScript una función puede escribirse de varias formas.

Función declarada clásica:

function sumar(a, b) {
  return a + b;
}

Función anónima asignada a una constante:

const sumar = function (a, b) {
  return a + b;
};

Función flecha con llaves y return explícito:

const sumar = (a, b) => {
  return a + b;
};

Función flecha compacta con retorno implícito:

const sumar = (a, b) => a + b;

Parte 2. Funciones anónimas

Ejercicio 2. Función clásica a función anónima

Código de partida:

function mostrarCurso() {
  console.log("DWCC");
}

Tarea:

  1. Reescribe la función como función anónima asignada a una constante llamada mostrarCurso.
  2. Llama a la función para comprobar que funciona.

Resultado esperado en consola:

DWCC

Ejercicio 3. Función anónima con parámetro

Crea una función anónima asignada a una constante llamada saludarAlumno.

La función debe recibir un parámetro nombre y mostrar por consola: 

Hola, Ana

Ejemplo de uso:

saludarAlumno("Ana");

Ejercicio 4. Función anónima con return

Crea una función anónima asignada a una constante llamada calcularIVA.

Debe recibir:

  • precio;
  • tipoIVA.

Debe devolver el importe del IVA, no mostrarlo directamente por consola.

Ejemplo:

const iva = calcularIVA(100, 21);
console.log(iva); // 21

Parte 3. Funciones flecha con llaves y return explícito

Ejercicio 5. Función clásica a función flecha

Código de partida:

function cuadrado(n) {
  return n * n;
}

Tarea:

  1. Reescribe la función como función flecha guardada en una constante llamada cuadrado.
  2. Usa la forma con llaves {} y return explícito.
  3. Prueba la función con:
console.log(cuadrado(5));

Resultado esperado:

25

Ejercicio 6. Función flecha con dos parámetros

Crea una función flecha llamada restar.

Debe recibir dos parámetros, a y b, y devolver la resta a - b.

Usa la forma con llaves y return explícito.

Ejemplo:

console.log(restar(10, 3)); // 7

Ejercicio 7. Función flecha con condicional

Crea una función flecha llamada esMayorDeEdad.

Debe recibir un parámetro edad y devolver:

  • true si edad >= 18;
  • false en caso contrario.

Usa la forma con llaves y return explícito.

Ejemplo:

console.log(esMayorDeEdad(20)); // true
console.log(esMayorDeEdad(15)); // false

Ejercicio 9. Función compacta con dos parámetros

Crea una función flecha compacta llamada multiplicar.

Debe recibir a y b, y devolver a * b.

Ejemplo:

console.log(multiplicar(4, 5)); // 20

Ejercicio 10. Función compacta con expresión booleana

Crea una función flecha compacta llamada esAprobado.

Debe recibir una nota y devolver directamente el resultado de comprobar si nota >= 5.

Ejemplo:

console.log(esAprobado(7)); // true
console.log(esAprobado(3)); // false

Parte 5. Elegir una forma adecuada

Ejercicio 11. ¿Con llaves o compacta?

Indica si usarías una función flecha compacta o una función flecha con llaves en cada caso. Justifica brevemente.

  1. Una función que recibe un número y devuelve su doble.
  2. Una función que recibe una nota y muestra varios mensajes por consola.
  3. Una función que recibe un precio, calcula descuento, calcula IVA y devuelve el total.
  4. Una función que recibe una edad y devuelve true o false.

Recuerda:

  • si solo devuelves una expresión sencilla, suele encajar bien la forma compacta;
  • si hay varias instrucciones, conviene usar llaves y return explícito cuando corresponda.

Parte 6. Callbacks básicos

Ejercicio 12. Pasar una función como argumento

Observa este código:

function saludar() {
  console.log("Hola desde la función saludar");
}

function ejecutar(funcionRecibida) {
  funcionRecibida();
}

ejecutar(saludar);

Responde:

  1. ¿Qué función recibe ejecutar como argumento?
  2. ¿En qué línea se llama realmente a saludar?
  3. ¿Qué se muestra por consola?
  4. ¿Por qué escribimos ejecutar(saludar) y no ejecutar(saludar())?

Ejercicio 13. Callback con operación matemática

Completa el siguiente código:

function calcular(a, b, operacion) {
  return operacion(a, b);
}

function sumar(x, y) {
  return x + y;
}

function multiplicar(x, y) {
  return x * y;
}

console.log(calcular(3, 4, sumar));       // ?
console.log(calcular(3, 4, multiplicar)); // ?

Tareas:

  1. Indica qué se muestra en cada console.log.
  2. Explica qué papel tiene el parámetro operacion.
  3. Explica por qué sumar y multiplicar son callbacks en este ejemplo.

Ejercicio 14. Callback con función anónima

Completa la llamada usando una función anónima que reste los dos números: 

const resultado = calcular(10, 4, function (x, y) {
  // completar

console.log(resultado); // 6

Parte 7. Pasar una función no es lo mismo que ejecutarla

Ejercicio 16. Detecta el error

Observa este código:

function mostrarMensaje() {
  console.log("Hola");
}

setTimeout(mostrarMensaje(), 2000);

Responde:

  1. ¿Qué problema tiene esta línea?
setTimeout(mostrarMensaje(), 2000);

  1. ¿Cuál sería la forma correcta?

  2. Explica con tus palabras la diferencia entre:

setTimeout(mostrarMensaje, 2000);

y:

setTimeout(mostrarMensaje(), 2000);

Ejercicio 17. setTimeout con función declarada

Crea una función llamada avisar que muestre:

Han pasado 2 segundos

Después, usa setTimeout para que se ejecute pasados 2000 milisegundos.

Pista:

setTimeout(avisar, 2000);

Ejercicio 18. setTimeout con función flecha inline

Reescribe este código usando una función flecha directamente dentro de setTimeout.

Código de partida:

setTimeout(function () {
  console.log("Mensaje después de 2 segundos");
}, 2000);
