B04. Práctica integradora: funciones en JavaScript

Un alto en el camino para consolidar funciones antes de avanzar hacia nuevos bloques de JavaScript.

1. Objetivo de la práctica

En esta práctica vas a construir un pequeño conjunto de funciones en JavaScript para resolver problemas sencillos. El objetivo no es hacer una aplicación grande, sino demostrar que sabes:

  • declarar funciones con function;
  • llamar funciones correctamente;
  • usar parámetros y argumentos;
  • devolver valores con return;
  • diferenciar console.log() de return;
  • usar variables locales;
  • combinar funciones con condicionales y bucles;
  • probar el código desde un fichero .js enlazado a una página HTML o desde la consola del navegador.

Esta práctica sirve para cerrar el bloque de funciones antes de empezar con arrays y objetos predefinidos.

2. Contenidos que se repasan

  • Variables y constantes.
  • Tipos básicos.
  • Conversión a número con Number().
  • Condicionales if, else if, else.
  • Operadores de comparación y operadores lógicos.
  • Bucles while y for.
  • Funciones básicas.
  • Parámetros, argumentos y return.
  • Ámbito local de las variables.

3. Normas de la práctica

  1. Usa un fichero llamado B04_practica_funciones.js.
  2. Puedes usar una página HTML mínima para enlazar el fichero JavaScript.
  3. No uses arrays todavía.
  4. No uses objetos propios todavía.
  5. En la parte obligatoria usa funciones declaradas con function.
  6. Usa console.log() para mostrar los resultados de las pruebas.
  7. Cada función debe hacer una tarea concreta.
  8. Evita meter todo el programa dentro de una sola función.

4. Estructura recomendada del fichero

Puedes organizar tu fichero así:

// B04_practica_funciones.js

// 1. Funciones de cálculo

// 2. Funciones con condicionales

// 3. Funciones con bucles

// 4. Pruebas de las funciones

La parte de pruebas debe ir al final del fichero, después de declarar las funciones.

5. Parte 1: funciones de cálculo

Ejercicio 1. Calcular el precio con IVA

Crea una función llamada calcularPrecioConIVA que reciba:

  • precioSinIVA
  • tipoIVA

La función debe devolver el precio final con IVA incluido.

Ejemplo de uso:

const total = calcularPrecioConIVA(100, 21);
console.log(total); // 121

Pistas:

const iva = precioSinIVA * (tipoIVA / 100);
const total = precioSinIVA + iva;

Ejercicio 2. Aplicar un descuento

Crea una función llamada aplicarDescuento que reciba:

  • precio
  • porcentajeDescuento

La función debe devolver el precio después de aplicar el descuento.

Ejemplo:

console.log(aplicarDescuento(100, 10)); // 90
console.log(aplicarDescuento(50, 20));  // 40

Ejercicio 3. Calcular precio final con descuento e IVA

Crea una función llamada calcularPrecioFinal que reciba:

  • precioBase
  • descuento
  • iva

La función debe:

  1. aplicar primero el descuento;
  2. aplicar después el IVA;
  3. devolver el precio final.

Debes reutilizar las funciones anteriores:

  • aplicarDescuento
  • calcularPrecioConIVA

Ejemplo:

const precioFinal = calcularPrecioFinal(100, 10, 21);
console.log(precioFinal); // 108.9

6. Parte 2: funciones con condicionales

Ejercicio 4. Saber si una nota está aprobada

Crea una función llamada estaAprobado que reciba una nota y devuelva:

  • true si la nota es mayor o igual que 5;
  • false en caso contrario.

Ejemplo:

console.log(estaAprobado(7)); // true
console.log(estaAprobado(3)); // false

Ejercicio 5. Obtener el texto de una calificación

Crea una función llamada obtenerTextoNota que reciba una nota de 0 a 10 y devuelva:

  • "Sobresaliente" si la nota es mayor o igual que 9;
  • "Notable" si la nota es mayor o igual que 7 y menor que 9;
  • "Aprobado" si la nota es mayor o igual que 5 y menor que 7;
  • "Suspenso" si la nota es menor que 5.

Ejemplo:

console.log(obtenerTextoNota(9));   // Sobresaliente
console.log(obtenerTextoNota(7.5)); // Notable
console.log(obtenerTextoNota(5));   // Aprobado
console.log(obtenerTextoNota(3));   // Suspenso

Ejercicio 6. Validar una nota

Crea una función llamada esNotaValida que reciba un valor y devuelva:

  • true si el valor es un número entre 0 y 10;
  • false si no lo es.

Debe controlar estos casos:

console.log(esNotaValida(7));       // true
console.log(esNotaValida(0));       // true
console.log(esNotaValida(10));      // true
console.log(esNotaValida(-1));      // false
console.log(esNotaValida(12));      // false
console.log(esNotaValida(NaN));     // false

Pista:

Number.isNaN(valor)

Ejercicio 7. Texto completo de una nota

Crea una función llamada generarMensajeNota que reciba una nota.

  • Si la nota no es válida, debe devolver "Nota no válida".
  • Si la nota es válida, debe devolver un texto como "Nota: 7. Resultado: Notable".

Debes reutilizar:

  • esNotaValida
  • obtenerTextoNota

Ejemplo:

console.log(generarMensajeNota(7));  // Nota: 7. Resultado: Notable
console.log(generarMensajeNota(12)); // Nota no válida

7. Parte 3: funciones con bucles

Ejercicio 8. Imprimir una tabla de multiplicar

Crea una función llamada imprimirTablaMultiplicar que reciba un número y muestre su tabla de multiplicar del 1 al 10.

Ejemplo:

imprimirTablaMultiplicar(5);

Salida esperada aproximada:

5 x 1 = 5
5 x 2 = 10
5 x 3 = 15
...
5 x 10 = 50

Esta función no necesita devolver un valor. Su tarea es mostrar información por consola.

Ejercicio 9. Sumar desde 1 hasta n

Crea una función llamada sumarHasta que reciba un número n y devuelva la suma desde 1 hasta n.

Ejemplo:

console.log(sumarHasta(5));  // 15, porque 1 + 2 + 3 + 4 + 5 = 15
console.log(sumarHasta(10)); // 55

Esta función sí debe usar return, porque necesitamos reutilizar el resultado.

Ejercicio 10. Contar números pares hasta n

Crea una función llamada contarParesHasta que reciba un número n y devuelva cuántos números pares hay desde 1 hasta n.

Ejemplo:

console.log(contarParesHasta(10)); // 5, porque son 2, 4, 6, 8, 10
console.log(contarParesHasta(7));  // 3, porque son 2, 4, 6

Pista:

if (i % 2 === 0) {
  // es par
}

8. Parte 4: miniaplicación final

Crea una función llamada mostrarResumenAlumno que reciba:

  • nombre
  • nota

La función debe mostrar en consola un resumen con este formato: 

Alumno/a: Ana
Nota: 7
Resultado: Notable
Estado: Aprobado

Condiciones:

  1. Debe usar esNotaValida para comprobar la nota.
  2. Si la nota no es válida, debe mostrar:
Alumno/a: Ana
Nota no válida
  1. Si la nota es válida, debe usar:
    • obtenerTextoNota
    • estaAprobado

Ejemplos de prueba:

mostrarResumenAlumno("Ana", 7);
mostrarResumenAlumno("Luis", 3);
mostrarResumenAlumno("Marta", 12);

9. Parte 5: revisión de errores frecuentes

Corrige los errores de los siguientes fragmentos.

Error 1. Falta llamar a la función

function saludar() {
  console.log("Hola");
}

saludar;

Pregunta: ¿por qué no se ejecuta la función?

Error 2. Usar una variable local fuera de la función

function calcularDoble(n) {
  const doble = n * 2;
  return doble;
}

console.log(doble);

Pregunta: ¿por qué falla console.log(doble)?

Error 3. Confundir console.log() con return 

function sumar(a, b) {
  console.log(a + b);
}

const resultado = sumar(3, 4);
console.log(resultado);

Pregunta: ¿qué valor tiene resultado? ¿Por qué?

Error 4. Código después de return

function ejemplo() {
  return "Fin";
  console.log("Esto se ejecuta");
}

console.log(ejemplo());

Pregunta: ¿se ejecuta el segundo console.log? Explica por qué.
