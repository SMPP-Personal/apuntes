Objetivo práctico

Qué vamos a conseguir hoy

Hoy no vamos a aprender HTML, CSS y JavaScript en profundidad. Vamos a comprobar que las tres piezas se conectan correctamente.
Crear un archivo index.html.
Crear una hoja de estilos css/estilos.css.
Crear un archivo JavaScript js/script.js.
Ver una página funcional en Chrome.
Mostrar un mensaje en la consola.
Detectar errores básicos de rutas.

Diagnóstico de ruta

Error preparado: el CSS no carga

Ruta incorrecta

<link rel="stylesheet" href="estilos.css">
→

Ruta correcta

<link rel="stylesheet" href="css/estilos.css">
El archivo está en css/estilos.css. La ruta debe coincidir con su ubicación real.

Diagnóstico final

Error final preparado

Opciones de error

  • Cambiar css/estilos.css por css/estilo.css.
  • Cambiar js/script.js por js/scripts.js.
  • Crear script.js fuera de la carpeta js.
  • No guardar antes de recargar.

Actividad

  1. Detecta el error.
  2. Explica qué falla.
  3. Corrígelo.
  4. Comprueba de nuevo.
