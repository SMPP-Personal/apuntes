Qué vamos a demostrar

Autonomía técnica

Crear una estructura de proyecto, separar archivos y trabajar desde una carpeta clara.

Comprobación real

Probar con Live Server, revisar Console, ejecutar Node.js y corregir errores preparados.

La práctica no mide una web espectacular. Mide autonomía técnica básica.

□ Trabaja dentro de la carpeta del proyecto
□ No uses archivos sueltos
□ Guarda antes de probar
□ Comprueba en Chrome
□ Abre Console
□ Corrige errores uno a uno
□ Haz commits con mensajes claros

const resultado = document.querySelector("#resultado");

console.log("Botón encontrado:", boton);
console.log("Zona de resultado encontrada:", resultado);

  console.log("Botón pulsado");
  resultado.textContent = "El proyecto está conectado correctamente.";
});
